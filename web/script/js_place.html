JavaScript Code 배치

<script> 매듭

HTML 문서에서 JavaScript Code는 <script> 와 </script> 매듭 사이에 작성한다. 

  <script>
    document.getElementById("id").innerHTML = "Hello, JavaScript...!";
  </script>

ECMAScript 2015 이전 JavaScript 에서는 형식 속성인 <script type = "text / javascript"> 를 사용하였지만, 현재는 JavaScript 가 HTML 문서의 기본 Script 언어이므로 형식 속성은 필요가 없다.

함수(Function)와 사건(Event)

JavaScript 함수는 사용자가 버튼을 클릭할 때와 같은 사건이 발생할 때 호출하여 실행될 수 있는 JavaScript 코드 묶음이다.

Script 배치

HTML 문서의 <head> 또는 <body> 매듭의 어디이든 원하는 수의 스크립트를 배치할 수 있다.

<head>의 JavaScript

  <!DOCTYPE html>
  <html>
  <head>
    <script>
        function change_paragaph() {
          document.getElementById("head").innerHTML = "Hello, JavaScript...!";
        }
    </script>
  </head>
  <body>
  
      <h1>자바스크립트/h1>
      <p id="head">안녕, 자바 스크립트...!</p>
      <button type="button" onclick="change_paragaph()">문단 내용 바꾸기</button>
  
  </body>
  </html>

Script 는 웹 브라우저에 의해서 순차적으로 해석되므로 <head> 요소 부분에 Script 가 있으면 Script 를 먼저 다운로드하게 되어 문서의 보여지는 속도가 느려진다. 그렇기 때문에 <body> 요소의 맨 아래에 Script 를 배치하면 표시 속도가 향상된다.

외부 JavaScript

Script 는 외부 파일에도 배치할 수 있다.

외부 파일 : extScript.js 
  function changeParagaph() {
    document.getElementById("para").innerHTML = "Hello, JavaScript...!"";
  }

  <script src="extScript.js"></script>

외부 Script 는 여러 개의 웹 문서에서 같은 코드를 사용할 수 있게 한다.
JavaScript 파일의 파일 확장자는 *.js 이다.
외부 Script 를 사용하려면 <script> 매듭의 src(source) 속성에 Script 파일의 이름을 지정한다.
외부 Script 는 <head>또는 <body> 부분의 어느 곳이든 원하는 대로 Script 참조를 배치할 수 있다.
외부 Script 는 <script> 매듭이 배치된 위치에 있는 것처럼 작동한다.

외부 Script 는 <script> 매듭을 포함할 수 없다.

외부 JavaScript 장점

  • HTML과 코드를 분리한다.
  • HTML과 JavaScript 를 읽고 유지 보수하기가 더 쉬워진다.
  • 먼저 내려받은 JavaScript 파일은 재사용되므로 문서 표시 속도를 높일 수 있다.

하나의 문서에 여러 Script 파일을 추가하려면 여러 개의 <script> 매듭을 사용해야 한다. 

  <script src="extScript_1.js"></script>
  <script src="extScript_2.js"></script>

외부 참조

외부 Script 는 전체 URL 또는 현재 웹 문서에 상대적인 경로로 참조될 수 있다.

전체 URL을 사용하여 Script 에 연결 

  <script src="https://gokgang.github.io/choi/script/js/extScript_1.js"></script>

현재 웹 사이트의 지정된 폴더(js)에 있는 Script 사용 

  <script src="js/extScript_1.js"></script>

현재 페이지와 같은 폴더에 있는 Script 사용 

  <script src="extScript_1.js"></script>
