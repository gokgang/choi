자료형

변수는 숫자, 문자열, 객체 등 다양한 자료형을 가질 수 있다 . 

var length = 16;
var lastName = "문재인";
var object = {familyName: "반", lastName: "기문"};

 // Number
// String
// Object

자료형의 개념

JavaScript의 기본 자료형(Primitive Data Type)은 숫자형(Number), 논리형(Boolean), 문자열(String), 기호문자(Symbol), null, undefined 가 있다.

숫자형(Number)

JavaScript 의 숫자형은 정수와 실수를 구분하지 않으며, 64 bit 부동 소수점의 자료이다. 숫자는 비트 0~51, 지수는 비트 52~62, 부호는 비트 63에 저장된다. 그 외에 기호 상수 Infinity(무한대), NaN(Not a Number) 이 있다. 

  <script>
    var x1 = 34.0;
    var x2 = 34;
    var x3 = 3.14;
    document.write("<br> x1 := " + x1 );
    document.write("<br> x2 := " + x2 );
    document.write("<br> x3 := " + x3 );
  </script>

문자형(String)

JavaScript 는 큰 따옴표(" "), 작은 따옴표(' '), 억음 부호(` `)에 묶여진 모든 글자와 숫자, 기호는 문자열로 평가한다.

문자열 더하기

숫자에 문자열을 더하기하면 숫자가 문자로 자동 형변환되어 문자열로 취급한다. 

  <script>
    var x = 16 + "Genesys";
    document.write(x);
  </script>

숫자와 문자열 더하기

  <script>
    var x = "16" + "Genesys";
    document.write(x);
  </script>

문자열과 문자열 더하기

  <script>
    var x = 16 + 4 + "Genesys";
    document.write(x);
  </script>

숫자와 숫자 그리고 문자열 더하기

  <script>
    var x = "Genesys" + 16 + 4;
    document.write(x);
  </script>

문자열과 숫자 그리고 숫자 더하기

JavaScript 의 변수는 동적으로 형변환이 된다.

  <script>
    var num;  // 자료형 미정
    document.write(num);
    document.write("<br>");
    num = 5;  // 숫자형(Number)
    document.write(num);
    document.write("<br>");
    num = "John"; // 문자형(String)
    document.write(num);
  </script>

문자열(String) 따옴표 안의 따옴표

문자형은 따옴표 안에 작성한다. 큰 따옴표 안에 작은 따옴표를 포함할 수 있으며, 그 반대로도 가능하다.

var answer1 = "문자형은 따옴표 안에 작성한다.";
var answer2 = "문자열을 둘러싼 따옴표와 일치하지 않는 한 '문자열 안에 따옴표'를 사용할 수 있다.";
var answer3 = '문자열을 둘러싼 따옴표와 일치하지 않는 한 "문자열 안에 따옴표"를 사용할 수 있다.';

논리형(Boolean)

논리형의 값은 true 와 false 두 가지 값이다. 

  <script>
  var x = 5;
  var y = 5;
  var z = 6;
    document.write(x == y);
    document.write("<br>");
    document.write(x == z);
  </script>

배열(Array)

배열은 대괄호로 작성하고, 배열 항목은 쉼표로 구분한다.

var cars = ["Saab", "Volvo", "BMW"];

배열의 순번은 0부터 시작한다. 즉, 첫 번째 항목은 [0], 두 번째 항목은 [1] 등이다.

객체(Object)

객체는 중괄호{ ... } 안에 쉼표로 구분된 이름 : 값의 쌍으로 작성한다.

const person = {familyName:"문", lastName:"재인", age:70, gender:"남자"};

typeof 연산자

typeof 연산자는 변수 또는 표현식의 자료형을 반환한다. 

  <script>
    document.write('typeof "" := ' + typeof "");
    document.write('<br>typeof "John" = ' + typeof "John");
    document.write('<br>typeof 0 := ' + typeof 0);
    document.write('<br>typeof 314 := ' + typeof 314);
    document.write('<br>typeof 3.14 := ' + typeof 3.14);
    document.write('<br>typeof (3 + 4) := ' + typeof (3 + 4));
  </script>

undefined 자료형

값이 없는 변수는 undefined 값을 갖는다. 자료형도 undefined 이다. 

  <script>
    var car;
    document.write("car := " + car);
    var car = undefined;
    document.write("<br>car := " + car);
    document.write("<br>typeof  car := " + typeof  car);
    var car = "";
    document.write("<br>typeof  car := " + typeof  car);
  </script>

null 자료형

JavaScript에서의 null 의 자료형은 객체(Object)이다 

  <script>
    var person = {familyName:"문", lastName:"재인", age:70, gender:"남자"};
    person = null; // 값은 null이 되지만 자료형은 객체이다.
    document.write("person := " + person);
    document.write("<br>typeof person : " + typeof person);
  </script>

다음과 같이 undefined 를 설정하여 객체를 비울 수도 있다. 

  <script>
    var person = {familyName:"문", lastName:"재인", age:70, gender:"남자"};
    person = undefined; // 값과 자료형이 모두 undefined 가 된다.
    document.write("person := " + person);
    document.write("<br>typeof person : " + typeof person);
  </script>

undefined 와 null 의 차이

  <script>
    document.write("typeof undefined := " + (typeof undefined));
    document.write("<br>typeof null := " + (typeof null));
    document.write("<br>(null === undefined) := " + (null === undefined));
    document.write("<br>(null == undefined) := " + (null == undefined));
  </script>

복합 자료(Complex Data)

typeof 연산자는 복합 자료에 대하여 객체(object)와 배열(array) 그리고 값이 null 인 경우 "object" 를 반환하고, 함수는 "function" 을 반환한다. 

  <script>
    document.write('typeof {name: "반기문", age: 70} := ' + typeof {name: "반기문", age: 70});
    document.write(' br>typeof [1, 2, 3, 4] := ' + typeof [1, 2, 3, 4]); 
    document.write(' br>typeof null := ' + typeof null);
    document.write(' br>typeof function fnuc(){} := ' + typeof function fnuc(){}); 
  </script>
