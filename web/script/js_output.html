JavaScript 출력

자료의 출력

  • innerHTML 을 사용하여 HTML 요소의 내용을 설정
  • document.write() 사용하여 HTML 문서에 직접 출력
  • window.alert() 을 사용하여 경고 상자에 표시
  • console.log() 를 사용하여 웹 브라우저 콘솔(Console)에 표시

HTML 요소의 내용 설정

JavaScript는 document.getElementById(id)메소드를 사용하여 HTML 요소에 접근한다.

innerHTML 속성은 id 속성을 가진 HTML 요소의 내용을 설정한다. 

  <p id="demo"></p>
  <script>
    document.getElementById("demo").innerHTML = 5 + 6;
  </script>

id="demo"

HTML 요소의 innerHTML 속성을 바꾸는 것은 HTML로 자료를 표시하는 일반적인 방법이다.

document.write () 사용

document.write()는 주로 테스트 목적으로 사용한다. 

  <script>
      document.write(5 + 6);
  </script>

주의 : HTML 문서가 로드된 후 document.write() 함수를 사용하면 기존 HTML이 모두 지워진다.

  <button type="button" onclick="document.write(5 + 6);">확 인</button>

document.write() 메서드는 글자만 사용해야 한다.

window.alert()의 사용

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <title>JavaScript Example</title>
</head>
<body>
  <h2>JavaScript Example</h2>

<script>
  window.alert(5 + 6);
</script>

</body>
</html>

window 키워드는 생략할 수 있다. JavaScript에서 window 객체는 전역 범위이므로 기본적으로 변수, 속성 및 메서드는 window 객체의 속한다. 따라서 window 키워드 없이 바로 사용할 수 있다.

console.log() 사용

브라우저에서 console.log() 메소드를 호출하여 JavaScript를 실행하여 디버깅할 수 있다.

웹 브라우저(Chrome)에서 지금 F12 키를 누르면 Console 창에서 "11"이 출력되는 것을 확인할 수 있다. 

<script>
  console.log(5 + 6);
</script>

JavaScript에서 인쇄하기

JavaScript에는 인쇄를 위한 개체 또는 메소드가 없으므로 직접 출력장치에 접근할 수는 없다.

그러나 브라우저의 window.print() 메소드를 호출하여 현재 창의 내용을 인쇄할 수 있다.

  <button onclick="window.print()">웹 문서 인쇄하기</button>
