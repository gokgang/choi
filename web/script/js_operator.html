덧셈 연산자( + )

두 개의 변수를 선언과 동시에 값을 할당하고, 하나의 변수는 선언과 동시에 앞서 선언한 두 변수의 합을 연산하여 할당한다.

  var x = 5;
  var y = 2;
  var z = x + y;

  // assign the value 5 to x
  // assign the value 2 to y
  // assogn the value 7 to z(5 + 2)

할당(Assign) 연산자

할당 연산자( = )는 변수에 값을 할당한다.

  var x = 10;

곱셈 연산자( * )

    var x = 5;
    var y = 2;
    var z = x * y;

덧셈 할당 연산자(Addition assignment operator)

  var x = 10;
  x += 5;

문자열 연산자(String Operators)

덧셈 + 연산자는 문자열을 합치는 데 사용할 수 있다.

  <script>
    var text1 = "John";
    var text2 = "Doe";
    var text3 = text1 + " " + text2;
    document.write(text3);
  </script>

+= 은 문자열 변수에 문자열을 추가한다.

  <script>
    var text1 = "What a very ";
    text1 += "nice day.";
    document.write(text1);
  </script>

+ 연산자는 문자열과 문자열을 연결한다.

문자열과 숫자의 더하기

숫자와 숫자를 더하면 합계를 반환하지만 숫자와 문자열을 더하면 문자열을 반환한다.

  <script>
    var x = 5 + 5;
    var y = "5" + 5;
    var z = "Hello" + 5;
    document.write("x := " + x);
    document.write(", y := " + y);
    document.write(", z := " + z);
  </script>

숫자와 문자열을 더하면 결과는 문자열이 된다!

연산자 설명

!= 같지 않다(not equal)

이진수 연산자(Bit Operator)

비트 연산자는 32비트 이진수 연산자이다. 연산 결과는 십진수로 반환한다.

연산자 설명 예제 예제 설명 결과 진수
& AND 5&1 0101 & 0001 0001 1
| OR 5|1 0101 | 0001 0101 5
~ NOT ~5 ~0101 1010 4
^ XOR 5^1 0101 ^ 0001 0100 4
<< Zero fill left shift 5<<1 0101 << 1 1010 10
>> Signed right shift 5>>1 0101 >> 1 0010 2
>>> Zero fill right shift 5>>>1 0101 >>> 1 0010 2

위의 예제는 4 비트 부호없는 예이다. 그러나 JavaScript는 32비트 부호있는 숫자를 사용한다.
