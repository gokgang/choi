JavaScript 객체

JavaScript 변수는 데이터 값을 담을 수 있는 그릇이다.

다음의 코드는 car라는 변수에 단순한 값을 저장한다.

  var car = "Genesys";
  document.getElementById("car").innerHTML = car;

객체는 많은 값을 가질 수 있는 변수이다.

다음의 코드는 car_type 이라는 변수에 많은 값을 할당한다.

    var car_type = {type:"Genesys", model:"GV80", color:"grey"};
    document.getElementById("genesys").innerHTML = 
      `형식 : ${car_type.type}, 모델 : ${car_type.model}, 색상 : ${car_type.color}`;
예제 결과 :

객체의 속성값은 Colon( : )으로 구분한 이름: 값 의 짝으로 작성한다.

JavaScript 객체는 속성(Property)과 기능(Methods)을 담을 수 있는 그릇이다.

객체 정의(Object Definition)

  // Create an object:
  var person = {familyName:"반", lastName:"기문", age:50};
  document.getElementById("person").innerHTML = 
    person.familyName + " " + person.lastName +  " " + person.age + "세";
예제 결과 :

객체의 정의는 여러 줄에 걸쳐 작성할 수 있다.

  // Create an object:
  var person = {
    familyName:"반",
    lastName:"기문",
    age:75
  };
  document.getElementById("obj_person").innerHTML = 
    person.familyName + " " + person.lastName +  " " + person.age + "세";

객체 속성(Object Properties)

JavaScript에서 객체의 이름: 값 의 짝을 속성(property)이라 한다.

속성(property)값(value)
familyName반
lastName기문
age75

객체 속성 접근(Accessing Object Properties)

객체의 속성에 접근하는 방법은 두 가지이다.

objectName.propertyName
or
objectName["propertyName"]

방법 1 : person.lastName;

  document.getElementById("exam1").innerHTML = person.lastName;

방법 2 : person["lastName"];

  document.getElementById("exam2").innerHTML = person["lastName"];

객체의 기능(Methods) 정의

객체는 여러가지 기능(Methods)을 가질 수 있다. 함수(Function)를 정의하는 방법으로 객체에서 실행할 기능을 정의한다.

속성값(value)
familyName반
lastName기문
age75
fullNamefunction() {return this.familyName + " " + this.lastName;}

기능(Methods)은 객체의 속성으로 포함된 함수이다.

  var obj_person = {
      familyName: "반",
      lastName: "기문",
      id: 5566,
      fullName: function() {
          return this.familyName + " " + this.lastName;
      }
  };

  document.getElementById("exam").innerHTML = obj_person.fullName();
Result :

this 지시어(Keyword)

객체의 기능 정의에서 this 는 객체 그 자신을 말한다.

위의 예에서 함수 정의에 사용된 this는 person 객체 그 자신이다.

즉, this.familyName은 person 객체의 familyName이다.

객체의 기능(Methods)에 접근

다음과 같이 객체의 기능에 접근한다.

objectName.methodName()
document.getElementById("ex").innerHTML = obj_person.fullName();

() 괄호없이 기능에 접근하면 기능의 정의 코드를 반환한다.

document.getElementById("ex2").innerHTML = obj_person.fullName;

이미 만들어져 있는 String, Number, Boolean 객체를 new 생성자로 객체를 다시 또 만들어서는 않된다.

var x = new String(); // Declares x as a String object

var y = new Number(); // Declares y as a Number object

var z = new Boolean(); // Declares z as a Boolean object

위와 같이 하면 코드가 복잡해 지고 실행속도가 느려진다.
