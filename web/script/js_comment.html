JavaScript 풀이(Comments)

JavaScript 풀이는 주석 또는 설명문이라고도 하며, JavaScript 코드를 설명하고 알아보기 쉽게 만드는 데 사용한다.
코드를 시험할 때 JavaScript 풀이를 사용하여 실행을 방지할 수도 있다.

한 줄 풀이(Comment)

한 줄 풀이는 두 줄 빗금(dubble slash, // )을 사용한다.
JavaScript는 맨 앞에 두 줄 빗금이 있는 줄의 문장은 무시하고 실행하지 않는다. 

  <script>
    // 제목 바꾸기
    document.getElementById("title").innerHTML = "JavaScript Comments";
    var x = 5;  // 변수 x를 선언하고, 5를 할당
    var y = x + 2;  // 변수 y를 선언하고, 변수 x의 값에 2를 더한 값을 할당
  </script>

여러 줄 풀이(comment block)

여러 줄 풀이는 /* 로 시작하고 */ 로 끝난다.
JavaScript 는 /* 와 */ 사이의 모든 글자는 무시하고 실행하지 않는다. 

  <script>
    /*
      다음은
      id="title" 인 제목을
      JavaScript Comments로 바꾼다.
      */
    document.getElementById("title").innerHTML = "JavaScript Comments";
  </script>

한 줄 풀이를 사용하는 것이 일반적이며, 여러 줄 풀이는 공식 문서에 주로 사용된다.

한 줄 풀이(comment)를 사용한 코드 실행 중지

  <script>
    //document.getElementById("title").innerHTML = "JavaScript Comments";
    document.getElementById("title").innerHTML = "자바 스크립트 풀이";
  </script>

여러 줄 풀이(comment block)를 사용한 코드 실행 중지

  <script>
    /*
    document.getElementById("title").innerHTML = "JavaScript Comments";
    document.getElementById("title").innerHTML = "자바 스크립트 풀이";
    */
  </script>
