JavaScript Events

JavaScript는 HTML 요소에서 발생하는 여러 가지 사건에 대하여 "반응"하게 할 수 있다.

    웹 문서의 사건 발생 예 :

  • 웹 페이지 로드 완료
  • 입력 필드의 값 변경
  • 사용자가 버튼을 클릭

HTML 요소의 사건 감지(event handler) 속성에 JavaScript 코드를 추가하여 사건이 발생했을 때 코드를 실행할 수 있다.

작은 따옴표를 사용한 구문

<element event= 'JavaScript Code'>

큰 따옴표를 사용한 구문

<element event= "JavaScript Code">

<button onclick="document.getElementById('demo').innerHTML = Date()">현재 시각 보기</button>
 

다음은 자체 요소의 내용 변경 :

<button onclick="this.innerHTML = Date()">현재 시각 보기</button>
 

일반적인 HTML 사건

Event설 명
onchange요소의 내용 변경
onclick요소를 클릭
onmouseover요소 위로 마우스를 이동
onmouseout요소에서 마우스가 나감
onkeydown키보드 키를 누름
onload브라우저가 페이지 로드를 완료

JavaScript의 기능

사건 감지기(Event Handlers) 를 사용하여 사용자 입력, 사용자 작업 및 브라우저 작업을 처리하고 확인할 수 있다.

  • 페이지가 로드될 때 실행하는 작업
  • 페이지를 닫을 때 해야 할 일
  • 사용자가 버튼을 클릭할 때 실행하는 작업
  • 사용자가 데이터를 입력할 때 확인해야 하는 내용

사건과 함께 작동하도록 하기 위해 다양한 방법을 사용할 수 있다.

  • HTML 요소의 사건 감지기 속성에 JavaScript 코드를 직접 작성할 수 있다.
  • 사건 감지기는 JavaScript 함수를 호출할 수 있다.
  • HTML 요소에 고유한 사건 처리기 함수를 할당할 수 있다.
  • 불필요한 사건 감지가 전송되거나 처리되는 것을 방지할 수 있다.
