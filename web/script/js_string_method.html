String Method

문자열 처리 방법 및 속성

"John Bidden"와 같은 기본값은 객체가 아니기 때문에 속성이나 메서드를 가질 수 없다.
그러나 JavaScript는 기본값을 객체로 취급하기 때문에 메서드와 속성도 기본값에 사용할 수 있다.

다음의 예제를 실습할 때의 CSS 설정을 다음과 같이 하세요. 

  <div style="border: 1px solid grey; padding: 0px 16px; white-space: pre-line;">
      <script>
        ... 
      </script>
  </div>

문자열 길이

length 속성은 문자열의 길이를 반환

  <script>
    var text = "나 보기가 역겨워 가실 때에는";
    var string_length = text.length;
    document.writeln(text);
    document.writeln("string_length := " + string_length);
  </script>

문자열에서 단어 찾기

indexOf() 메서드는 문자열에서 지정된 단어가 있는 첫 번째 글자의 순번(index)을 반환한다. 순번은 항상 0 부터 시작한다. 

  <script>
    var text = "영변의 약산 진달래 꽃";
    var position = text.indexOf("진달래");
    document.writeln(text);
    document.writeln(" '진달래' 위치 := " + position)
  </script>

lastIndexOf() 메서드는 문자열에서 지정된 단어의 마지막으로 발생한 순번을 반환한다. 

  <script>
    var text = "영변의 약산 진달래 꽃, 영변의 약산 진달래 꽃";
    var position = text.lastIndexOf("진달래");
    document.writeln(text);
    document.writeln("마지막 '진달래' 위치 := " + position)
  </script>

indexOf() 와 lastIndexOf() 메서드는 문자열에서 지정된 단어를 찾지 못하면 -1 을 반환한다. 

  <script>
    var text = "영변의 약산 진달래 꽃, 영변의 약산 진달래 꽃";
    var position = text.lastIndexOf("진달래꽃");
    document.writeln(text);
    document.writeln(" '진달래꽃' 위치 := " + position)
  </script>

두 번째 매개변수가 있으면 그 위치에서 부터 검색한다. 

  <script>
    var text = "영변의 약산 진달래 꽃, 영변의 약산 진달래 꽃";
    var position = text.indexOf("진달래", 15);
    document.writeln(text);
    document.writeln(" '진달래' 위치 := " + position)
  </script>

  <script>
    var text = "영변의 약산 진달래 꽃, 영변의 약산 진달래 꽃";
    var position = text.lastIndexOf("진달래", 15);
    document.writeln(text);
    document.writeln("16번째부터 마지막  '진달래' 위치 := " + position)
  </script>

search() 메서드

  <script>
    var text = "영변의 약산 진달래 꽃, 영변의 약산 진달래 꽃";
    var position = text.search("진달래");
    document.writeln(text);
    document.writeln("'진달래' 위치 := " + position);
  </script>

search() 메서드는 두 번째 시작 위치 인수를 사용할 수 없지만 정규식을 사용할 수 있다.
indexOf() 메서드는 두 번째 시작위치 인수를 사용할 수 있지만 정규식을 사용할 수 없다.

문자열 일부 잘라내기

  • slice (start, end)
  • substring (start, end)
  • substr (start, length)

slice () 메서드

slice() 메서드는 문자열의 일부를 추출하하여 문자열로 반환한다.
이 메서드는 시작 위치와 끝 위치의 두 개의 매개 변수를 사용한다.
끝 위치를 생략하면 나머니 모든 문자열을 추출한다. 

  <script>
    var text = "아름따다 가실 길에 뿌리오리다";
    var str = text.slice(5, 9);
    document.writeln(text);
    document.writeln("5 ~ 9 까지의 글 := " + str);
  </script>

  <script>
    var text = "아름따다 가실 길에 뿌리오리다";
    var str = text.slice(5);
    document.writeln(text);
    document.writeln("5 번 이후의 모든 글 :=" + str);
  </script>

매개 변수가 음수이면 문자열의 끝부터 위치를 계산한다. 

  <script>
    var text = "아름따다 가실 길에 뿌리오리다";
    var str = text.slice(-12, -6);
    document.writeln(text);
    document.writeln("-12 ~ -6 까지의 글 :=  :=" + str);
  </script>

두 번째 매개 변수를 생략하면 나머지 문자열을 모두 잘라낸다. 

  <script>
    var text = "아름따다 가실 길에 뿌리오리다";
    var str = text.slice(5);
    document.writeln(text);
    document.writeln("5번째 이후의 모든 글 :=  :=" + str);
  </script>

끝부터 계산 

  <script>
    var text = "아름따다 가실 길에 뿌리오리다";
    var str = text.slice(-6);
    document.writeln(text);
    document.writeln("끝에서 6번째 부터 모든 글 :=  :=" + str);
  </script>

Internet Explorer 8 및 이전 버전에서 음수는 작동하지 않는다.

substring () 메서드

substring() 은 slice() 과 유사하지만 순번에 음수를 사용할 수 없다. 

  <script>
    var text = "아름따다 가실 길에 뿌리오리다";
    var str = text.substring(5, 10);
    document.writeln(text);
    document.writeln("5 ~ 10 까지의 글 :=  :=" + str);
  </script>

두 번째 매개 변수를 생략하면 나머지 문자열을 모두 잘라낸다. 

  <script>
    var text = "아름따다 가실 길에 뿌리오리다";
    var str = text.substring(5);
    document.writeln(text);
    document.writeln("5 번째에서 나머지 모든 글자 :=  :=" + str);
  </script>

substr () 메서드

substr() 은 slice() 와 비슷하지만 두 번째 매개 변수에 추출할 글자수를 지정한다 

  <script>
    var text = "아름따다 가실 길에 뿌리오리다";
    var str = text.substr(5, 8);
    document.writeln(text);
    document.writeln("5 번째에서 부터 8개의 글자 :=  :=" + str);
  </script>

두 번째 매개 변수를 생략하면 나머지 모두를 잘라낸다. 

  <script>
    var text = "아름따다 가실 길에 뿌리오리다";
    var str = text.substr(5);
    document.writeln(text);
    document.writeln("5 번째에서 나머지 모든 글자 :=  :=" + str);
  </script>

첫 번째 매개 변수가 음수이면 문자열의 끝에서 부터 계산한다. 

  <script>
    var text = "아름따다 가실 길에 뿌리오리다";
    var str = text.substr(-5);
    document.writeln(text);
    document.writeln("끝에서 5 번째에서 나머지 모든 글자 :=  :=" + str);
  </script>

문자열 내용 바꾸기

replace() 메서드는 지정된 값을 문자열의 다른 값으로 바꾼다. 

  <script>
    var str = "Hello JavaScript!";
    var msg = str.replace("Hello", "안녕");
    document.writeln(str);
    document.writeln(msg);
  </script>

replace() 메서드는 원래의 문자열을 변경하지는 않고 새로운 문자열을 반환한다.
replace() 메서드는 일치하는 첫 번째 단어만 바꾼다. 또한 영문 대-소문자를 구분한다.
대-소문자를 구분하지 않고 바꾸려면 정규표현식에 /i 를 함께 사용하면 된다. 정규표현식은 따옴표없이 작성한다. 

      <script>
        var str = "Hello JavaScript!";
        var msg = str.replace(/HELLO/i, "안녕");
        document.writeln(str);
        document.writeln(msg);
      </script>

모든 일치 항목을 바꾸려면 정규 표현식에 /g (전역 일치) 와 함께 사용 

      <script>
        var str = "Hello JavaScript! Hello?";
        var msg = str.replace(/Hello/g, "안녕");
        document.writeln(str);
        document.writeln(msg);
      </script>

소문자를 대문자로 변환 toUpperCase()

  <script>
    var text = "Hello JavaScript!";
    document.writeln(text.toUpperCase());
  </script>

대문자를 소문자로 변환 toLowerCase()

  <script>
    var text = "Hello JavaScript!";
    document.write(text.toLowerCase());
  </script>

문자열 결합 concat()

  <script>
    var text1 = "Hello";
    var text2 = "JavaScript!";
    var text3 = text1.concat(" ", text2);
    document.write(text3);
  </script>

다음 두 개의 문장은 같은 의미이다.

var text = "Hello" + " " + "JavaScript!"; var text = "Hello".concat(" ", "JavaScript!");

모든 문자열 조작 메서드는 원래 문자열을 바꾸지 않으며, 새로운 문자열을 반환한다. 원래 문자열은 변경할 수 없으며 대체만 가능하다.

문자열 양쪽 공백 제거 trim()

  <script>
    var str = "       Hello World!        ";
    alert(str.trim());
  </script>

trim() 은 Internet Explorer 8 이하에서 지원되지 않는다.
IE 8을 지원해야 하는 경우 replace() 에 정규식을 사용하여 해결할 수 있다. 

  <script>
    var str = "       Hello World!        ";
    alert(str.replace(/^[\s\uFEFF\xA0]+|[\s\uFEFF\xA0]+$/g, ' '));
  </script>

String.prototype 으로 트림 기능 추가

  <script>
    if (!String.prototype.trim) {
      String.prototype.trim = function () {
        return this.replace(/^[\s\uFEFF\xA0]+|[\s\uFEFF\xA0]+$/g, ' ');
      };
    }
  </script>

JavaScript 문자열 채우기 padStart, padEnd

ECMAScript 2017 은 문자열의 시작 (padStart) 과 문자열의 끝 (padEnd) 에 채우기를 할 수 있도록 했다. 

  <script>
    var str = "5";
    str = str.padStart(4, 0);
    // Result is 0005
    document.write(str);
  </script>

  

  

  <script>
    var str = "5";
    str = str. padEnd(4, "＊");
    // Result is 5000
    document.write(str);
  </script>

Internet Explorer 에서는 문자열 채우기를 지원하지 않는다.

글자 뽑아내기

문자열에서 글자를 뽑아내는 방법

  • charAt ( position )
  • charCodeAt ( position )
  • 속성에 접근 []

charAt ()

charAt() 메서드는 문자열의 지정된 위치에 있는 글자를 반환한다. 

  <script>
    var str = "Hello World";
    document.write (str.charAt (6)); // return W
  </script>

charCodeAt ()

charCodeAt() 메서드는 문자열의 지정된 위치에 있는 문자의 유니 코드를 반환한다.
이 메서드는 0에서 65535 사이의 정수로 UTF-16 코드를 반환한다.
참고 : 영문자는 한 글자는 8비트를 사용하고 한글은 한 글자는 16비트를 사용한다. 

  <script>
    var str = "대한민국";
    document.write (str.charCodeAt (1)); // return 
  </script>

문자열의 속성 접근

ECMAScript 5 (2009) 부터 대괄호 [index] 를 사용하여 배열처럼 문자열의 속성에 접근할 수 있도록 허용한다. 

  <script>
    var str = "Hello JavaScript!";
    document.write(str[6]); // return J
  </script>

속성 접근 방법은 예측 불가능할 수 있다.

  • Internet Explorer 7 또는 이전 버전에서는 작동하지 않는다.
  • 문자열을 배열처럼 보이게 하지만 문자열은 배열이 아니다.
  • 문자가 없으면 [index] 는 undefined 를 반환하고 string.charAt () 은 빈 문자열을 반환한다.
  • 읽기 전용이므로 string [0] = "A" 와 같은 문장은 오류가 발생하지 않지만 작동하지 않는다!

  <script>
    var str = "Hello JavaScript!";
    str[0] = "A"; // no error
    document.write(str[0]); // return H
  </script>

문자열을 배열로 변환

split() 메서드를 사용하여 문자열을 배열로 변환할 수 있다. 

  <script>
    var text = "대,한,민,국"; // String
    var strArray = text.split(",");  // split on commas
    document.write(strArray[0]);
  </script>

  

  <script>
    var text = "대 한 민 국"; // String
    var strArray = text.split(" ");  // split on spaces
    document.write(strArray[1]);
  </script>

  

  <script>
    var text = "대|한|민|국"; // String
    var strArray = text.split("|");  // split on pipe
    document.write(strArray[3]);
  </script>

많이 사용하는 방법

  <script>
    var text = "대한민국 대통령 문재인"; // String
    var strArray = text.split(" ");  // split on spaces
    document.writeln(strArray[0]);
    document.writeln(strArray[1]);
    document.write(strArray[2]);
  </script>

구분 기호를 생략하면 반환된 배열에는 순번 [0]의 전체 문자열 배열이 된다. 

  <script>
    var text = "대한민국 대통령 문재인"; // String
    var arrString = text.split();  // no split
    document.write(arrString[0]);
  </script>

구분 기호가 "" 인 경우 반환된 배열은 단일 문자열의 배열이 된다. 

  <script>
    var text = "대한민국 대통령 문재인"; // String
    var arrString = text.split("");  // no split
    document.writeln(arrString[0]);
    document.writeln(arrString[1]);
    document.writeln(arrString[2]);
    document.write(arrString[3]);
  </script>
