할당 연산자(Assignment Operators)

연산자예제풀이
=x = yx = y
+=x += yx = x + y
-=x -= yx = x - y
*=x *= yx = x * y
/=x /= yx = x / y
%=x %= yx = x % y
<<=x <<= yx = x << y
>>=x >>= yx = x >> y
>>>=x >>>= yx = x >>> y
&=x &= yx = x & y
^=x ^= yx = x ^ y
|=x |= yx = x | y
**=x **= yx = x ** y
**= 연산자는 ECMAScript 2016 에 추가되었다.

할당 연산자

  var x = 10;

덧셈 할당 연산( += ) :

  var x = 10;
  var x += 5;

뻴셈 할당 연산( -= ) :

  var x = 10;
  var x -= 5;

곱셈 할당 연산( *= ) :

  var x = 10;
  var x *= 5;

나눗셈 할당 연산( /= ) :

  var x = 10;
  var x /= 5;

나머지 할당 연산( %= ) :

  var x = 10;
  var x %= 5;
