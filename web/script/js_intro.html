JavaScript 개요

HTML 요소의 내용 바꾸기

id="greetings" 인 요소를 찾아 요소의 내용을 "Hellow, JavaScript...!"로 설정한다. 

  <script>
    document.getElementById('greetings').innerHTML = "Hello JavaScript...!";
  </script>

JavaScript는 큰 따옴표(" ")와 작은 따옴표(' ')를 모두 허용한다.

HTML 요소의 속성(src) 값 바꾸기

  <button onclick="document.getElementById('bulb').src = 'image/img_bulbon.gif'">Turn On</button>
  <img id="bulb" src="image/img_bulboff.gif" style="width: 80px;">
  <button onclick="document.getElementById('bulb').src = 'image/img_bulboff.gif'">Turn Off</button>

HTML 요소의 style 속성 값 바꾸기

다음 예제는 <div> 요소의 바탕색을 바꾼다.

  <script>
    document.getElementById('background').style.backgroundColor  = 'skyblue';
  <script>

HTML 요소 숨기기 / 보이기

style 속성의 display 값을 바꾸어 HTML 요소를 숨기기한다.

  <p id="id_p">JavaScript는 이 글을 담고 있는 HTML 요소를 감출 수 있다.</p>
  <button onclick="document.getElementById('id_p').style.display = 'none'">글 감추기</button>
  <button onclick="document.getElementById('id_p').style.display = 'block'">글 보이기</button>

JavaScript는 이 글을 담고 있는 HTML 요소를 감출 수 있다.

JavaScript는 1995 년 Brendan Eich에 의해 발명되었으며 1997년에 ECMA 표준이되었다.
JavaScript의 공식 이름은 ECMAScript 이다.
ECMAScript 버전은 ES1, ES2, ES3, ES5, ES6 로 줄여서 부른다.
2016년 이후 새로운 버전은 연도별로 이름이 지정된다 (ECMAScript 2016/2017/2018).

Internet Explorer 는 ECMAScript 2015를 지원하지 않으며, 다른 모든 브라우저 들은 이를 지원한다. 2015 이후의 최신 버전은 아직 지원하지 않는 브라우저가 있으므로 너무 최신 버전을 사용하지 않는 것이 좋다.
