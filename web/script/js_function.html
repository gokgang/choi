JavaScript 함수

함수는 특정 작업을 수행하도록 설계된 코드 묶음이다. 함수는 호출되었을 때 비로소 실행된다.

수학에서 함수는 다음과 같이 계산한다.

함수 : f(x) = x + 1

y = f(x)

x = 2 이면

y = f(2) = 2 + 1

따라서 y = 3 이다.

이 함수를 JavaScript의 문법으로 정의하면 다음과 같다.

  function f(x) {return x +1;}
    var x = 2;
  if (x == 2) var y = f(x);

함수 정의 구문(Syntax)

JavaScript 함수는 function 지시어, 함수이름, 괄호(), 실행문 순으로 정의한다.

함수이름은 문자, 숫자, 밑줄 및 달러기호가 포함될수 있으며, 이는 변수이름과 규칙이 같다.

괄호 안에는 쉼표로 구분하여 매개변수 이름을 포함할 수 있다.(예: parameter1, parameter2, ... )

함수에서 실행할 문장은 중괄호 {...} 묶음 안에 배치한다.

    function name(parameter1, parameter2, parameter3) {
        // 실행문
    }

함수의 매개변수(parameter)는 함수 정의에서 괄호 안에 나열한다.

함수 호출시 인수값이 있으면 함수에의해 처리된다.

함수 내에서 인수(매개변수)는 지역변수이다.

함수는 다른 프로그래밍 언어에서 프로시져 또는 서브 루틴과 동일하다.

함수 호출(Function invocation)

함수 안의 실행문은 함수를 호출(Invokes, calls)하였을 때 비로소 실행된다.

  • 버튼 클릭과 같은 사건(event) 발생시
  • JavaScript 코드에서 호출
  • 자동(자체 호출) 실행

함수의 반환값(Function return value)

JavaScript가 return 문에 도달하면 함수의 실행이 중지된다.

함수가 다른 명령문에서 호출된 경우 JavaScript는 호출 명령문 다음의 코드를 실행한다.

함수는 반환값을 연산하여 호출한 곳에 결과값을 반환한다.

다음은 두 숫자의 곱을 계산하여 결과값을 반환한다.

  <script>
    //함수 호출, 반환 값을 product 에 할당
    var product = multiply(4, 3);
    // 함수는 매개변수 a와 b의 곱셈 결과 반환
    function multiply(a, b) {
      return a * b;
    }
    document.write("product := " + product);
  </script>

함수의 기능

한번 정의한 코드를 여러번 재사용할 수 있다.

인수를 다르게 함수를 호출하면 다른 결과를 얻을 수 있다.

화씨를 섭씨로 변환 : 

  function toCelsius (fahrenheit) {
      return (5 / 9) * (fahrenheit - 32);
  }

() 연산자는 함수를 호출한다

위의 예제를 사용하여 toCelsius()로 함수를 호출하고 함수의 결과를 참조한다.

  document.write("toCelsius(50) := " + toCelsius(50));

함수의 반환값을 변수의 값으로 사용

함수는 모든 유형의 공식, 할당 및 계산에서 변수를 사용하는 것과 같은 방식으로 사용할 수 있다.

함수의 반환값을 저장하는 변수를 사용하는 대신 : 

    var toC = toCelsius(77);
    var text = "온도는 섭씨 " + toC + " 도";

함수를 변수값으로 직접 사용할 수 있다.

var text = "온도는 섭씨 " + toCelsius(77) + " 도";

지역 변수(Local Variables)

JavaScript의 함수 안에서 선언된 변수는 지역변수(Local variables)이다.

지역변수는 함수 안에서만 사용할 수 있다.

  // carName 변수는 여기에서 사용 불가
  function localFunction() {
      var carName = "Genesys";
      // 여기에서는 carName 변수 사용 가능
      document.getElementById('demo1').innerHTML = typeof carName;
      return carName;
  }
  document.write("carName : = " + localFunction());
  // carName 변수는 여기에서 사용 불가
  document.getElementById('demo2').innerHTML = typeof carName;

지역변수는 함수 내에서만 인식되기 때문에 같은 이름의 변수를 다른 함수에서 사용할 수 있다.

지역변수는 함수가 시작될 때 생성되고 함수가 완료되면 소멸한다.
