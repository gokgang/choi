산술 연산

일반적인 산술 연산은 두 개의 상수(leteral)를 대상으로 연산을 한다.

var x = 100 + 50;

JavaScript에서 변수를 사용하여 산술 연산을 할 수 있다.

var x = a + b;

표현식(Expression)으로 연산

var x = (100 + 50) * a;

연산자(Operators)와 피연산자(Operands)

산술 연산에서 숫자를 피연산자라고 한다. 두 피연산자 사이에서 수행되는 연산은 연산자에 의해 결정된다.

피연산자 연산자 피연산자

덧셈 연산(Adding)

덧셈 연산자(Addition operator)는 ( + )를 사용한다.

  var x = 5;
  var y = 2;
  var z = x + y;

뻴셈 연산(Subtracting)

뻴셈 연산자(Subtraction operator)는 ( - )를 사용한다.

  var x = 5;
  var y = 2
  var z = x - y;

곱셈 연산(Multiplying)

곱셈 연산자(Multiplication operator)는 ( * )를 사용한다.

  var x = 5;
  var y = 2;
  var z = x * y;

나눗셈 연산(Dividing)

나눗셈 연산자(Division operator)는 ( / )를 사용한다.

var x = 5;
  var y = 2;
  var z = x / y;

나머지 연산(Remainder)

나머지 연산자(Modulus operator)는 ( % )를 사용한다.

  var x = 5;
  var y = 2;
  var z = x % y;

산술 연산에서 두 정수를 나누면 몫과 나머지가 있다.

수학에서 나머지 연산(Modulo operation)의 결과는 나눗셈의 나머지이다.

증가 연산(Increment)

증가 연산자( ++ )는 수치를 증가시킨다.

  var x = 5;
  x++;
  var z = x;

감소 연산(Descripting)

감소 연산자( -- )는 수치를 감소시킨다.

  var x = 5;
  x--;
  var z = x;

지수 연산(Exponentiation)

지수 연산자( ** )는 수치의 지수 연산을 한다.

  var x = 5;
  var z = x ** 2; // result is 25

x ** y 는 Math.pow(x, y)와 같은 결과를 얻는다

  var x = 5;
  var z = Math.pow(x, 2); // result is 25

연산자 우선 순위(Operator Precedence)

연산자는 산술 표현식에서 연산이 수행되는 순서가 있다.

  var x = 100 + 50 * 3;

위의 예제에서 수학에서 처럼 곱셈이 먼저 계산된다.

곱하기( * ) 및 나누기( / )는 더하기( + ) 와 뻬기( - ) 보다 우선 순위가 높다.

그리고 수학에서와 같이 괄호()를 사용하여 우선 순위를 변경할 수 있다.

  var x = (100 + 50) * 3;

괄호를 사용하면 괄호 안의 연산이 먼저 계산된다.

많은 연산자가 동일한 우선순위를 갖는 경우 왼쪽에서 오른쪽으로 계산된다.

  var x = 100 + 50 - 3;
