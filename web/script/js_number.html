JavaScript 숫자형

숫자 정확도 문자열 덧셈 문자열 숫자 NaN 무한대 16 진수 숫자 객체

JavaScript에는 한 가지의 숫자형(Number) ​​만 있다. 숫자형은 정수와 소수의 구분없이 모두 쓸 수 있다. 

  <script>
    var num = 3.14; // a number with decimals
    document.writeln("num := " + num);
    var int = 3;    // a number without decimals
    document.write("int := " + int);
  </script>

매우 크거나 매우 작은 숫자는 과학적 지수 표기법으로 작성할 수 있다. 

  <script>
    var num = 123E-5; // 0.00123 
    document.writeln("num := " + num);
    var int = 123e5;    // 12300000 
    document.write("int := " + int);
  </script>

숫자는 항상 64 비트 부동 소수점

다른 많은 프로그래밍 언어와 달리 JavaScript는 정수, short, long, 부동 소수점 등과 같은 다른 유형의 숫자를 구분하지 않는다.
JavaScript 숫자는 국제 IEEE 754 표준에 따라 항상 배정밀도 부동 소수점 숫자로 저장된다.
이 형식은 64 비트로 숫자를 저장한다. 숫자는 비트 0 ~ 51( 52bits ), 지수는 비트 52 ~ 62( 11bits ), 부호는 비트 63( 1bit )에 저장된다.

정확도

소숫점 또는 지수 표기법이 없는 정수는 최대 15 자리까지 정확도를 갖는다.

<script> var maxNum = 999999999999999; // maxNum will be 999999999999999 var overflowNum = 9999999999999999; // overflowNum will be 10000000000000000 var maxNum =999999999999999; var overflowNum = 9999999999999999; document.writeln("maxNum := " + maxNum); document.write("overflowNum := " + overflowNum); </script>

최대 소수점 수는 17 개이지만 부동 소수점 산술이 항상 100 % 정확한 것은 아니다. 

    <script>
      let x = 0.2 + 0.1;  // 연산결과 x 의 값 0.30000000000000004
      document.write (" x := " + x);
    </script>

위와 같은 문제는 다음과 같이 해결한다. 

    <script>
      let x = (0.2  * 10 + 0.1 * 10) / 10;  // 연산결과 x 의 값 0.3
      document.write (" x := " + x);
    </script>

숫자와 문자열 덧셈

JavaScript 는 더하기와 문자열 연결에 모두 + 연산자를 사용한다.

두 개의 숫자를 더하면 결과는 숫자

  <script>
    let x = 10;
    let y = 20;
    let sum = x + y;  // sum 은 숫자 30 할당
    document.write("sum := " + sum);
  </script>

두 개의 문자열을 더하면 결과는 문자열 연결 

  <script>
    let x = "10";
    let y = "20";
    let sum = x + y;  // sum 은 글자 1020 할당
    document.write("sum := " + sum);
  </script>

숫자에 문자열을 더하면 결과는 문자열 

  <script>
    let x = 10;
    let y = "20";
    let sum = x + y;  // sum 은 글자 1020 할당
    document.write("sum := " + sum);
  </script>

문자열에 숫자를 더하면 결과는 문자열 

  <script>
    let x = "10";
    let y = 20;
    let sum = x + y;  // sum 은 글자 1020 할당
    document.write("sum := " + sum);
  </script>

다음과 같은 일반적인 실수는 이 연산 결과가 30 일 것으로 예상하는 것 

  <script>
    let x = 10;
    let y = 20;
    let result = "연산 결과 : " +  x + y;
    document.write(result);
  </script>

다음과 같은 일반적인 실수는 이 연산 결과가 102030이 될 것으로 예상하는 것 

  <script>
    let x = 10;
    let y = 20;
    let z = "30";
    let result =  x + y + z;
    document.write(result);
  </script>

JavaScript 해석기(interpreter)는 왼쪽에서 오른쪽으로 작동한다.
x와 y가 모두 숫자이기 때문에 처음 10 + 20 = 30이 된다.
다음으로 z는 문자열이기 때문에 30 + "30"으로 문자열 연결이 된다.

문자열 숫자

JavaScript 문자열은 숫자를 내용에 포함할 수 있다. 모든 숫자 연산에서 문자열을 숫자로 변환하려고 한다. 

  <script>
  let x = "100";
  let y = "10";
  let result = x / y;   // result 10
    document.write("result := " + result);
  </script>

이것 또한 가능 

  <script>
  let x = "100";
  let y = "10";
  let result = x * y;   // result 1000
    document.write("result := " + result);
  </script>

이것 또한 가능 

  <script>
  let x = "100";
  let y = "10";
  let result = x - y;   // result 90
    document.write("result := " + result);
  </script>

이것은 문자열 연결 

  <script>
  let x = "100";
  let y = "10";
  let result = x + y;   // result 10010
    document.write("result := " + result);
  </script>

NaN - 숫자 아님

NaN 은 숫자가 아님을 나타내는 JavaScript 예약어이다.
숫자가 아닌 문자열로 산술 연산을 하려고 하면 NaN (Not a Number)이 된다. 

  <script>
  let num = 100 / "Apple";  // result NaN(Not a Number)
    document.write("num := " + num)
  </script>

문자열에 숫자 값이 포함된 경우 결과는 숫자 

  <script>
  let num = 100 / "10";  // result 10
    document.write("num := " + num)
  </script>

JavaScript 전역 함수 isNaN() 을 사용하여 값이 숫자인지 확인 

  <script>
  let num = 100 / "Apple";  // num is NaN(Not a Number)
  document.write("isNaN(num) := " + isNaN(num));  // return true
  </script>

주의 

  <script>
  let x = "NaN";
  let y = "2";
  let result = x + y;   // result NaN
    document.write("result := " + result);
  </script>

  <script>
  let x = "NaN";
  let y = "2";
  let result = x + y;   // result NaN2
    document.write("result := " + result);
  </script>

NaN 은 숫자 자료형 

  <script>
    document.write(" typeof NaN := " +  typeof NaN);
  </script>

무한대 (Infinity or -Infinity)

  <script>
    let letNum = 2;
    while (letNum != Infinity) {  // 무한대가 될 때까지 실행
      letNum = letNum * letNum;
      document.writeln(letNum);
    }
  </script>

숫자를 0으로 나누면 무한대( Infinity ) 

  <script>
    document.writeln("2 / 0 := " +( 2 / 0));  // x will be Infinity
    document.write("-2 / 0 := " +(- 2 / 0));  // y will be -Infinity
  </script>

Infinity 은 숫자 자료형 

  <script>
    document.write(" typeof Infinity := " +  typeof Infinity);
  </script>

16 진수( Hexadecimal )

JavaScript는 숫자 상수 앞에 0x가 있는 경우 16 진수로 해석한다. 

  <script>
    let x = 0xFF;
    document.write("0xFF := " + x);
  </script>

07 과 같이 숫자 앞에 0 을 붙이지 말 것.
일부 JavaScript 버전에서 숫자 앞에 0 이 있으면 8 진수로 해석할 수 있다.

기본적으로 JavaScript는 숫자를 10 진수로 표시한다. 10진수 이외의 숫자를 표시하려면 toString() 함수를 사용하여 나타낼 수 있다. 

  <script>
    let myNumber = 32;
    document.writeln("myNumber.toString(10) := " + myNumber.toString(10));  // returns 32
    document.writeln("myNumber.toString(32) := " + myNumber.toString(32));  // returns 10
    document.writeln("myNumber.toString(16) := " + myNumber.toString(16));  // returns 20
    document.writeln("myNumber.toString(8) := " + myNumber.toString(8));  // returns 40
    document.writeln("myNumber.toString(2) := " + myNumber.toString(2));  // returns 100000
  </script>

숫자 객체

JavaScript 숫자는 실효값으로 생성된 기본값이다.

숫자를 new 지시어를 사용하여 객체로 정의할 수도 있다. 

  <script>
      let num = 123;
      let obj_num = new Number(123);
      document.writeln("typeof num := " + typeof num );
      document.write("typeof obj_num := " + typeof obj_num );
  </script>

new 지시어로 Number 개체를 만들면 코드가 복잡해지고 실행속도가 느려지며, 예상하지 못한 결과가 발생할 수도 있다.

논리 연산자 "=="로 값의 비교

  <script>
    {
      let num = 500;
      let obj_num = new Number(500);
      document.write("(num == obj_num) := " + (num == obj_num));
    }
  </script>

논리 연산자 "==="로 값과 자료형 모두 비교

  <script>
    {
      let num = 500;
      let obj_num = new Number(500);
      document.write("(num === obj_num) := " + (num === obj_num));
    }
  </script>

객체 끼리는 비교할 수 없다.

  <script>
    {
      let obj_n = new Number(500);
      let obj_m = new Number(500);
      document.write("결과 := " + (obj_n == obj_m));
    }
  </script>

"==" 연산자는 값을 비교하고, "===" 연산자는 값과 자료형을 모두 비교하며, 객체와 객체를 비교하면 항상 "false" 이다.
