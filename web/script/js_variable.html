JavaScript 변수

변수는 컴퓨터의 메모리 내에서 독특한 이름을 가진 데이터 값을 저장하기 위한 영역을 말한다.

변수의 이름을 프로그램 내에서 사용하면 실제로 해당 메모리 영역에 저장된 데이터를 사용할 수 있게 된다.

다음은 var 키워드로 x, y, sum 세 개의 변수를 선언하고 초기값을 할당한다. 

  var x = 5;
  var y = 6;
  var sum = x + y;

    위의 예에 대한 설명

  • 변수 x 를 선언하고 값 5를 저장한다.
  • 변수 y 를 선언하고 값 6를 저장한다.
  • 변수 sum 을 선언하고 x와 y의 값을 더한 값을 저장한다.

새로운 변수 선언 예약어 "const" 와 "let"

2015년 이전에는 var 예약어를 사용하는 것이 JavaScript에서 변수를 선언하는 유일한 방법이었다.

그러나 JavaScript 2015 vesion(ES5-ECMAScript 2015)에서는 const 지시어는 재 할당할 수 없는 상수를 정의하고, let는 범위가 제한된 변수를 정의할 수 있다.

변수의 연산

  var x = 5;
  var y = 6;
  var sum = x + y;

수학에서와 마찬가지로 값을 유지하기 위해 변수(예: x, y, sum)를 사용한다.

수학에서와 마찬가지로 수식에 변수를 사용한다(예: sum = x + y).

위의 예서 sum은 11로 평가된다.

JavaScript 식별자(Identifiers)

식별자는 길이의 제한이 없으며, 의미있는 이름을 사용하는 것이 좋다.

변수 이름의 일반적 규칙

  • 문자, 숫자, 밑줄( _ ) 및 달러 기호( $ )가 포함될 수 있다.
  • 첫글자는 문자 또는 달러 기호( $ ), 밑줄( _ )로 시작해야 한다.
    • (달러 기호( $ )와 밑줄( _ )은 특별한 경우에 첫 글자로 사용하므로 가급적 사용하지 않는 것이 좋다.)
  • 대·소문자를 구분한다.
  • 예약어(Reserved Words)는 이름으로 사용할 수 없다(예 : JavaScript 지시어).

JavaScript 식별자(Identifiers)는 대·소문자를 구분한다.

할당 연산자(Assignment Operator)

JavaScript에서 등호( = )는 "같다(equal to)"는 의미가 아니라 "할당(assignment)" 연산자이다.

다음은 수학에서와는 다르게 변수 x의 현재 값에 5를 더한 값을 x에 저장한다는 의미이다.

x = x + 5

JavaScript에서 "같다(equal to)"는 의미의 연산자는 "==" 이다.

자료형(Data types)

JavaScript 변수는 100과 같은 숫자와 "John"과 같은 글자들을 모두 저장할 수 있다.

Progrmming에서 글자들을 문자열이라고 한다.

JavaScript는 다양한 유형의 자료를 처리할 수 있지만 지금은 숫자와 문자열만 다루기로 한다.

문자열은 큰 따옴표( " " ) 또는 작은 따옴표( ' ' ) 안에 작성한다. 숫자는 따옴표없이 작성한다.

따옴표 안에 숫자가 있으면 문자열로 처리된다. 

  var pi = 3.14;
  var person = "John Bidden";  
  var answer = 'Yes I am!';

JavaScript 변수 선언(variable declaration)

JavaScript에서 변수를 생성하도록 하는 것을 변수 "선언(declaration)"이라 한다.

다음은 var 지시어를 사용하여 JavaScript 변수를 선언한다.

var carName;

변수를 선언한 후 변수에는 값이 없다.(JavaScript 용어: undefined)

변수에 값을 할당하려면 " = " 기호(Assigned symbol)를 사용한다.

carName = "Volvo";

변수를 선언할 때 바로 값을 할당할 수도 있다.

var carName = "Volvo";

다음의 예제에서는 변수 carName을 생성하고 "Volvo" 값을 할당한다.

그 다음 id="demo" 인 HTML 요소의 값으로 할당한다.

  <p id="demo"></p>
  <script>
      var carName = "Volvo";
      document.getElementById("demo").innerHTML = carName;
  </script>
<p id="demo">

</p>

Script 시작 부분에 모든 변수를 선언하는 것은 좋은 프로그래밍 방법이다.

한 줄에 여러 개의 변수 선언

한 줄에 쉼표로 구분하고 여러 개의 변수를 선언할 수 있다.

var person = "John Doe", carName = "Volvo", price = 200;

선언을 여러 줄에 걸쳐 할 수도 있다.

  var person = "John Doe",
  carName = "Volvo",
  price = 200;

변수의 초기값

선언한 변수는 초기값을 할당하기 전에는 예약어 undefined값을 갖는다.

다음 예제의 carName 변수는 선언된 후 undefined값을 갖는다.

var carName;

변수 다시 선언

먼저 선언되어 있는 변수를 다시 선언하면 원래 있던 값이 없어지지 않는다.

다음의 예제는 변수 carName을 다시 선언해도 변수값은 그대로이다. 

  <script>
    var carName = "Volvo";
    var carName;
    document.write("carName := " + carName);
  </script>

산술 연산

JavaScript는 = 및 + 연산자를 사용하여 산술 연산을 실행한다.

    var x = 5 + 2 + 3;
    document.write("x := " + x);

문자열 더하기 연산

    var name = "John" + " " + "Bidden";
    document.write("name : " + Bidden);

또 다른 예제

    var x = "5" + 2 + 3;

따옴표 안에 숫자를 넣으면 나머지 숫자는 문자열로 처리되어 연결된다.

또 다른 예제

    var x =  2 + 3 + "5";
    document.write("x := " + x);

달러 기호 $

JavaScript는 달러 기호를 문자로 취급하므로 $를 포함하는 식별자는 유효한 변수 이름이다.

  var $$$ = "Hello World";
  var $ = 2;
  var $myMoney = 5;

달러 기호를 사용하는 것은 JavaScript에서 일반적이지 않지만 전문 프로그래머는 JavaScript 라이브러리의 주 함수에 대한 별칭으로 사용한다.

예를 들어 JavaScript 라이브러리 jQuery에서 메인 함수 $는 HTML 요소를 선택하는데 사용된다. jQuery에서 $("p");는 "모든 p 요소 선택"을 의미한다.

밑줄( _ )

JavaScript는 밑줄을 문자로 취급하므로 _ 를 포함하는 식별자는 유효한 변수 이름이다.

  var _lastName = "Johnson";
  var _x = 2';
  var _100 = 5;

밑줄을 사용하는 것은 JavaScript에서 그다지 일반적이지 않지만, 전문 프로그래머는 "개인(숨긴)" 변수의 별칭(alias)으로 사용하는 것이 관례이다.
