숫자 처리 방법(Number Methods)

3.14 또는 12345와 같은 실효값(Primitive values)은 객체가 아니기 때문에 속성(properties)과 처리 방법(Methods)을 가질 수 없다. 그러나 JavaScript는 실효값도 객체로 취급하기 때문에 속성과 처리 방법을 사용할 수 있다.

toString() 처리 방법(Method)

toString() 처리 방법은 숫자를 문자열로 반환한다. 

  <script>
    let num = 12345;
    let str = ""
    str += num.toString(); // 변수 num에서 문자열 "12345" 를 반환한다.
    str += "<br>";
    str += (6789).toString();  // 실효값 6789 에서 문자열 "6789" 를 반환한다.
    str += "<br>";
    str += (50000 + 4321).toString();  // 식 50000 + 4321 에서 문자열 "54321" 을 반환한다.
    str += "<br>";
    document.write(str);
  </script>

toExponential() 처리 방법(Method)

toExponential() 은 지수 표기법을 사용하여 반올림하여 기록한 숫자가 있는 문자열을 반환한다. 매개 변수는 소수점 뒤의 자릿 수를 정의한다. 지정하지 않으면 반올림하지 않는다. 

  <script>
  let num = 9.656;
  let str = "";
    str += num.toExponential(); // 9.656e+0 반환
    str += "<br>";
    str += num.toExponential(2); // 9.66e+0 반환
    str += "<br>";
    str += num.toExponential(4); // 9.6560e+0 반환
    str += "<br>";
    str += num.toExponential(6); // 9.656000e+0 반환
    str += "<br>";
    document.write(str);
  </script>

toFixed () 처리 방법(Method)

toFixed() 는 지정된 소수 자릿수로 쓰여진 숫자가 있는 문자열을 반환한다.
