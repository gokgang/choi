문자열

문자열은 글자를 저장하고 조작하는 데 사용된다.

JavaScript 문자열

JavaScript 문자열은 따옴표로 묶여진 글자, 숫자, 기호 등이다. 따옴표는 작은 따옴표 또는 큰 따옴표를 모두 사용할 수 있다. 문자열을 둘러싼 따옴표가 같지 않으면 문자열 안에 따옴표를 사용할 수 있다. 다시 말해 큰 따옴표 안에서 작은 따옴표를 사용할 수 있으며, 작은 따옴표 안에서 큰 따옴표를 사용할 수 있다.

var str = "이 글은 큰 따옴표 안에서 '작은 따옴표'를 사용하는 예문이다."
var str = '이 글은 작은 따옴표 안에서 "큰 따옴표"를 사용하는 예문이다.'

문자열 길이

문자열의 길이는 내장 속성 length 를 사용하여 구한다. 

  <script>
    var text = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
    document.write("text.length := " + text.length);
  </script>

  <script>
    var text = "0123456789";
    document.write("text.length := " + text.length);
  </script>

  <script>
  // 공백도 하나의 문자
    var text = "나 보기가 역겨워 가실 때에는";
    document.write("text.length := " + text.length);
  </script>

도피( escape ) 문자

문자열은 따옴표로 묶어야 하므로 따옴표를 사용할 수 없다. 그래서 따옴표를 사용하려면 backslash ( \ ) 문자를 따옴표 앞에 추가한다.

표기결과설 명
\''Single quote
\""Double quote
\\\Backslash 
  var str = "이 글은 문자열에 \"따옴표\" 삽입 예문입니다."
  document.write(str);

기타 도피(escape) 문자

다음의 이스케이프 문자는 원래 타자기, 텔레타이프 및 팩스기를 제어하기 위해 설계되었다. HTML에서는 별 의미가 없다.

표기결과
\bBackspace
\fForm Feed
\nNew Line
\rCarriage Return
\tHorizontal Tabulator
\vVertical Tabulator

참고 : <pre> 요소의 매듭을 사용하거나 CSS 를 white-space: pre; white-space: pre-line; white-space: pre-wrap; 을 지정한 경우에는 개행문자( \n )를 사용할 수 있다. 

    <div style="white-space: pre-line; border: 1px solid grey;">
      <script>
        document.write("Hello World\n");
        // writeln()은 write()에 개행문자를 추가한 것이다
        document.writeln("Hello World");
      </script>
    <div>

코드 줄 바꾸기

코드를 알기 쉽게 하기 위하여 일반적으로 코드의 길이를 80자 이내로 작성한다. 문장이 한 줄을 넘는 경우 줄을 바꾸기 가장 좋은 곳은 연산자의 뒤이다. 

  <script>
    document.getElementById('id_0').innerHTML =
    "Hello JavaScript!";
    </script>

문자열의 중간에서 줄바꿈을 하려면 하나의 backslash( \ )로 나눌 수 있다. 

  <script>
    document.getElementById('id_1').innerHTML = "Hello \
      JavaScript!";
    </script>

이 방법은 좋은 방법이 아니며 지원되지 않을 수도 있다. 일부 브라우저는 backslash( \ ) 문자 뒤에 공백을 허용하지 않는다.

문자열을 분리하는 안전한 방법은 문자열에 덧셈( + ) 연산자를 사용하는 것이다. 

      <script>
        document.getElementById('id_2').innerHTML = "Hello "+
          "JavaScript!";
        </script>

backslash 로 코드 줄을 나눌 수는 없다. 

  <script>
    document.getElementById('id_3').innerHTML = \
    "Hello JavaScript!";
    </script>

문자열 객체

JavaScript 문자열은 실효값(leteral)으로 만들어진 기본값이다.
그러나 new 지시어를 사용하여 String 객체로 만들 수도 있다.

var familyname = new String("John");

문자열을 객체로 만들면 실행속도가 느려지고 코드를 복잡하게 만들며 예기치 않은 결과가 생길 수 있다.

문자열과 문자열 객체의 차이

두 변수의 값 비교

  <script>
    var str_name = "John";
    var obj_name = new String("John");
    document.write ("(str_name == obj_name) := " + (str_name == obj_name)) ;
  </script>

두 변수의 값과 자료형 비교

  <script>
    var str_name = "John";
    var obj_name = new String("John");
    document.write ("(str_name === obj_name) := " + (str_name === obj_name)) ;
  </script>

객체는 서로 비교 불가

  <script>
    var obj_name01 = new String("John");
    var obj_name02 = new String("John");
    document.write ("(obj_name01 === obj_name02) := " + (obj_name01 === obj_name02)) ;
  </script>
