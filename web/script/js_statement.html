JavaScript 문장(statements)

JavaScript 프로그램은 웹 브라우저에 의해 실행된다.
컴퓨터에 의해 실행할 명령어의 목록을 문(statements)이라 한다.

JavaScript 문장의 구성

JavaScript 문장은 실효값(Literal Values), 연산자(Operators), 표현식(Expressions), 지시어(Keywords) 및 주석(Comments) 등으로 구성되며, 작성된 순서대로 실행된다.

다음은 id = "greetings"인 HTML 요소 안에 "Hello JavaScript!"를 넣도록 지시한다. 

    document.getElementById("greetings").innerHTML = "Hello JavaScript!";

문장의 구분

JavaScript 의 문장은 줄 바꿈으로 구분한다.
지시어(Keyword)와 단어(Token) 사이는 공백(White Space)으로 구분하고,
단어(Token)와 단어 사이는 쉼표( , )로 구분하며,
문장(Statement)과 문장은 Semiconlon( ; )으로 구분하는 것을 권장한다. 

  <script>
    var x, y, sum;
    x = 5;
    y = 6;
    sum = x + y;
  </script>

  
    // 문장 1
    // 문장 2
    // 문장 3
    // 문장 4

문장을 세미콜론으로 구분하면 같은 줄에 여러 문장을 배치할 수 있다. 

  var x, y, sum; x = 5; y = 6; sum = x + y;
하나의 문장을 여러 줄에 걸쳐서 작성할 수 있다. 
  <script>
    var x = 
    8
    document.write(
      "x := " + x
    )
  </script>

공백

JavaScript는 공백을 무시한다. 스크립트에 공백을 추가하여 알아보기 쉽게 만들 수 있다.

다음의 두 줄은 동일하다.

  var person = "John";
  var person="John";

연산자 ( = + - * / ) 주위에는 공백을 두는 것이 좋다.

  var sum = x + y;

줄 길이 및 줄 바꿈

알아보기 쉽게 하기 위하여 한 줄의 길이가 80자 보다 긴 코드는 피하는 것이 좋다.

한 줄의 길이가 긴 경우에 연산자 뒤에서 나누는 것이 가장 좋다. 

    document.getElementById("greetings").innerHTML = 
    "Hello JavaScript!";

코드 묶음

JavaScript 문장은 중괄호 {...} 안에 코드 묶음을 만들 수 있다.

코드 묶음의 목적은 함께 실행될 문장을 한 묶음에 정의한다.

다음의 setConents() 함수는 함께 실행할 명령문의 묶음으로 만든다. 

  <script>
    function setConents() {
      document.getElementById('greetings_1').innerHTML = "안녕하세요?";
      document.getElementById('greetings_2').innerHTML = "반갑습니다!";
    }

    setConents();
  </script>

id="greetings_1"

id="greetings_2"

JavaScript 지시어(Keyword)

JavaScript의 문장은 지시어로 시작하는 경우가 많다.

다음은 여기에서 사용할 지시어들이다. 이것은 지시어의 일부이며 보다 더 많은 지시어가 있다.

예약어(Keyword)설명(Description)
break switch문 또는 loop 종료
continue loop에서 다시 맨 위에서 시작
debugger 실행을 중지하고 디버깅 기능을 호출(가능한 경우)
do...while 조건이 참이면 계속 블록 실행
for 조건이 참이면 계속 블록 실행
function 함수 선언
if...else 조건에 따라 명령문을 선택적으로 실행
return 함수 종료
switch 조건에 따라 명령문을 선택하여 실행
try...catch 블록 안의 오류 처리
var 변수 선언

JavaScript의 지시어(Keywords)는 예약어(Reserved Words)이다.
예약어는 변수(Variables)의 이름으로 사용할 수 없다.
