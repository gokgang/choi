바깥 띄우기(Margins)

margin 은 정의된 테두리 바깥의 요소 주위에 띄우기를 지정한다.

이 요소의 바깥 띄우기는 margin: 70px 로 지정되었다.

따로 따로 바깥 띄우기

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

margin 의 속성값

  • auto : right 와 left 를 같은 값으로 설정
  • length : px, pt, cm 등으로 띄움 값 지정
  • % : 포함하고 있는 상위 요소의 백분율로 띄우기 지정
  • inherit : 상위 요소로 부터 상속

    • 참고 : 음수 값 허용

    <style>
      div {
        margin-top: 50px;
        margin-right: 100px;
        margin-bottom: 150px;
        margin-left: 200px;
        border: 1px solid grey;
      }
    </style>

줄임 속성 (margin)

위와 같이 따로 따로 속성을 지정할 수 있지만 하나로 단축한 줄임 속성인 margin 을 사용하여 코드를 줄일 수 있다.

줄임 속성의 설정 순서는 top, right, bottom, left 순으로 시계 회전 방향으로 설정값을 나열한다. 4개의 속성값 가운데 생략된 것이 있으면 서로 마주 보는 쪽의 값으로 지정된다. 설정값이 3개이면 순서대로 지정되고 마지막의 left 는 마주보는 값인 right 와 같은 값으로 지정된다. 2개이면 top 과 bottom, right 와 left 가 같은 값을 갖게 된다. 설정값이 하나이면 당연히 모두 같은 값을 갖게 된다.

바깥 띄우기 자동

margin: auto로 설정하면 지정된 너비를 가진 요소가 차지하고 남은 좌우의 바깥 띄우기를 같은 값으로 설정한다. 따라서 요소가 가운데 정렬이 된다.

브라우저의 좌우 크기를 조절해 보면 예제의 사각형이 항상 가운데에 위치함을 확인할 수 있다. 

  <style>
    div {
      width: 300px;
      margin: auto;
      border: 1px solid red;
    }
  </style>

설정값 상속

다른 요소에 포함된 요소는 상위 요소로 부터 설정값을 상속받을 수 있다. 

  <style>
    div {
      border: 1px solid red;
      margin-left: 100px;
    }
    p.ex {
      margin-left: inherit;
    }
  </style>

이 문단은 왼쪽 바깥 띄우기를 div 요소로 부터 상속받는다.

바깥 띄우기 줄임

한 줄을 모두 차지하는 요소와 요소 사이의 아래 위로 바깥 띄우기(margin)가 설정되어 있으면 설정값이 큰 값이 적용된다. 반면, 같은 줄에 있는 요소 사이의 바깥 띄우기는 설정값 대로 지정된다. 이를 바깥 띄우기 줄임(margin collapse) 이라고 한다. 

  <style>
    h1 {
      margin: 0 0 50px 0;
    }
    h2 {
      margin: 20px 0 0 0;
    }
  </style>

위와 같이 바깥 띄우기가 설정된 경우 margin: 50px 가 적용된다.
