CSS에서는 RGB, HEX, HSL, RGBA, HSLA 값을 사용하여 색상을 지정할 수 있다.

RGB 색상

RGB 색상 값은 RED, GREEN 및 BLUE 광원을 나타낸다.

CSS에서는 다음과 같이 색상을 RGB 값으로 지정할 수 있다.

rgb(red, green, blue)

매개 변수 (빨간색, 녹색, 파란색)는 0에서 255 사이의 색상 강도를 지정한다.
rgb (255, 0, 0)는 빨간색이 가장 높은 값 (255)으로 설정되고, 나머지는 0으로 설정되어 있으므로 빨간색으로 표시된다.
검정을 표시하려면 rgb (0, 0, 0)과 같이 모든 색상 매개 변수를 0으로 설정한다.
흰색을 표시하려면 rgb (255, 255, 255)와 같이 모든 색상 매개 변수를 255로 설정한다.

RGB (255, 0, 0)

RGBA 색상

RGBA 색상 값은 색상의 불투명도를 지정하는 알파 값이 있는 RGB 색상 값의 확장이다.
alpha 매개 변수는 0.0 (완전 투명) ~ 1.0 (불투명) 사이의 숫자.

rgba(red, green, blue, alpha)

RGBA (255, 0, 0, 0.5)

16진수(HEX) 색상

16 진수 색상은 b #RRGGBB 로 지정된다.
RR (빨간색), GG (녹색), BB (파란색)의 16 진수 정수는 색상의 구성 요소를 지정한다.
#FF0000은 빨간색이 가장 높은 값 (FF)으로 설정되고, 나머지는 가장 낮은 값 (00)으로 설정되므로 빨간색으로 표시된다.

#FF0000

3 자리 HEX 값

3 자리 16 진수값은 6 자리 16 진수 코드의 줄임이다.
#RGB 는 0과 F 사이의 값을 가진 빨강, 녹색 및 파랑 구성 요소를 나타낸다.
3 자리 16 진수 코드는 두 값 (RR, GG, BB)이 각 구성 요소에 대해 같은 경우에만 사용할 수 있다.
따라서 #FF00CC가 있으면 #F0C와 같이 작성할 수 있다.

#F0C

HSL 색상

HSL은 색조(hue), 채도(saturation), 밝기(lightness를 나타낸다.

HSL (hue, saturation, lightness)

색조(hue)는 0에서 360까지 색상환의 각도이다. 0은 빨간색, 120은 녹색, 240은 파란색이다.
채도(saturation는 백분율 값, 0 %는 회색 음영, 100 %는 전체 색상이다.
밝기(lightness도 백분율, 0 %는 검은 색, 50 %는 중간, 100 %는 흰색이다.

HSL(0, 100%, 50%)

HSLA 색상

HSLA 색상 값은 색상의 불투명도를 지정하는 알파 값이 있는 HSL 색상 값의 확장이다.

hsla(hue, saturation, lightness, alpha)

alpha 값이 0.0 이면 완전 투명이고, 1.0 이면 불투명하게 된다.

HSLA (0, 100%, 50%, 0.5)
