CSS 선택자(Selectors)

CSS 선택자는 양식(Style)을 지정할 HTML 요소를 선택한다.

● 요소(Elements) 선택자 : HTML 요소 이름으로 선택
● 단순(Simple) 선택자 : 이름(name), 식별자(id), 무리(Class)를 기준으로 요소 선택
● 조합(Combinators) 선택자 : 특정 관계에 따라 요소 선택
● 유사 무리(Pseudo-classes) 선택자 : 특정 상태에 따라 비슷한 요소 선택
● 유사 요소(Pseudo-elements) 선택자 : 요소의 일부 선택 및 비슷한 양식(Style) 선택
● 속성(attribute) 선택자 : 속성 또는 속성 값에 따라 요소 선택

요소 선택자

요소 이름으로 양식을 지정할 HTML 요소를 선택한다.

문서 안의 모든 <p> 요소를 빨간 색 글자에 가운데 맞춤으로 설정

  <style>
    p {
      text-align: center;
      color: red;
    }
  </style>

id 선택자

HTML 요소의 id 속성을 가진 특정 요소를 선택한다.
요소의 id 는 문서 안에서 고유하므로 id 선택자는 하나의 고유한 요소를 선택하는 데 사용한다.
특정 id 를 가진 요소를 선택하려면 id 앞에 hash( # ) 기호를 붙여서 작성한다.

id="para"인 요소의 양식을 지정

  <style>
  #para {
    text-align: center;
    color: red;
  }
</style>

참고 : 식별자 id 의 첫 글자는 숫자로 시작할 수 없다

무리(Class) 선택자

무리(Class) 선택자는 HTML 요소의 특정 class 속성을 가진 무리를 선택한다.
특정 무리의 요소를 선택하려면 무리의 이름 앞에 마침표 ( . ) 기호를 붙여서 작성한다.

class = "center"인 모든 HTML 요소는 빨간색이고 가운데 맞춤으로 지정

  <style>
    .center {
      text-align: center;
      color: red;
    }
  </style>

특정 HTML 요소 만 무리의 영향을 받도록 지정할 수도 있다.

class = "center"인 <p> 요소 만 빨간색으로 가운데 맞춤 

  <style>
    p.center {
      text-align: center;
      color: red;
    }
  </style>

HTML 요소는 둘 이상의 무리를 참조할 수도 있다.

<p> 요소는 class = "center"및 class = "large"에 따라 양식 지정

<p class="center large">이 문단은 두 무리(class)의 양식을 참조한다.</p>

무리(class) 이름의 첫 긑자는 숫자로 시작할 수 없다.

CSS 범용(universal) 선택자 ( * )

범용 선택자 ( * )는 문서의 모든 HTML 요소를 선택한다.

다음의 선언은 문서의 모든 HTML 요소에 영향을 준다. 

  <style>
    * {
      text-align: center;
      color: red;
    }
  </style>

선택자 묶음

같은 양식을 정의한 모든 선택자를 묶음으로 양식을 지정할 수 있다.

다음 CCS 는 모두 같은 양식을 지정하고 있다. 

  <style>
    h1 {
      text-align: center;
      color: red;
    }

    h2 {
      text-align: center;
      color: red;
    }

    p {
      text-align: center;
      color: red;
    }
  </style>

선택자를 쉼표로 구분하여 한 묶음으로 양식을 지정하면 코드를 최소화할 수 있다. 

  <style>
    h1, h2, p {
      text-align: center;
      color: red;
    }
  </style>

단순 선택자

선택자예 :설명
#id #firstname id="firstname"인 요소 선택
.class .intro class="intro"인 무리를 모두 선택
* * 모든 요소 선택
요소 p 모든 <p> 요소 선택
요소1, 요소2, ... div, p 모든 <div> 요소와 <p> 요소 선택
