CSS 구문(Syntax)

CSS 규칙은 선택자(Selector)와 선언(Declaration)으로 구성된다.

선택자선언선언
h1{color:blue;font-size:12px;}
 ￪￪￪￪
 속성값속성값

선택자는 CSS 속성을 지정할 HTML 요소를 가르킨다.

선언은 세미콜론으로 구분된 하나 이상의 선언을 포함한다.

선언은 CSS 속성 이름과 값을 콜론으로 구분한다.

여러 개의 CSS 속성은 세미콜론으로 구분되며 선언 묶음은 중괄호로 묶는다.

예 :

다음 예는 모든 <p> 요소가 빨간색으로 가운데 정렬된다.

  <style>
    p {
      color: red;
      text-align: center;
    }
  </style>

설명 :

  • p는 선택자(Selector) (CSS를 지정할 HTML 요소: <p>)
  • color는 속성, red 는 속성값
  • text-align는 속성, center 는 속성값
