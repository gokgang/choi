글(text) 서식

이 텍스트는 일부 서식 속성으로 양식이 지정된다. 제목은 가운데 정렬, 텍스트 변환 및 색상 속성을 사용한다. 단락이 들여 쓰기되고 정렬되며 문자 사이의 공백이 지정됩니다. 색상이 지정된 이 링크에는 밑줄이 제거된다

글자색

color 속성은 글자색을 설정하는 데 사용한다. 색상값은 다음과 같이 지정한다.

  • 색상이름 - 예) red
  • HEX 값 - 예) #ff0000
  • RGB 값 - 예) rgb(255, 0, 0)

웹 문서의 기본 양식은 선택자 body 에 정의한다. 

  <style>
    body {
      background-color: lightgrey;
      color: blue;
    }
    h1 {
      background-color: grey;
      color: yellow;
    }
  </style>

글 정렬

text-align 속성은 글의 가로 정렬을 설정한다. 글은 왼쪽(left), 오른쪽(right), 가운데(center), 양쪽 맞춤(justify)이 가능하다. 글의 방향(direction)이 왼쪽에서 오른쪽이면 왼쪽 정렬이 기본값이고, 글의 방향이 오른쪽에서 왼쪽이면 오른쪽 정렬이 기본값이다. 

    h1 {
      text-align: center;
    }

Heading 1 - text-align: center;

    h2 {
      text-align: left;
    }

Heading 2 - text-align: left;

    h3 {
      text-align: right;
    }

Heading 3 - text-align: right;

정렬 속성이 justify 로 설정된 경우 각 줄이 확장되어 모든 줄이 같은 너비를 가지며, 왼쪽 및 오른쪽이 가지런하게 정렬된다. 

    p {
      border: 1px solid grey;
      padding: 16px;
      width: 200px;
      height: 220px;
      text-align: justify;
    }

글 꾸미기

text-decoration 속성은 글에 장식을 설정하거나 설정된 장식을 제거하는 데 사용한다.

다음은 <a> 요소에 설정된 밑줄을 제거한다. 

  a {
    text-decoration: none;
  }

text-decoration 속성은 글에 장식을 설정한다 

  span.overline {
    text-decoration: overline;
  }
  span.linethrough {
    text-decoration: line-through;
  }
  span.underline {
    text-decoration: underline;
  }

글자 전환

text-transform 속성은 글자를 대문자와 소문자를 지정하는 데 사용된다. 모든 글자를 대문자 또는 소문자로 바꾸거나 각 단어의 첫 글자를 대문자로 바꾸는 데 사용할 수 있다. 

  <style>
    p.uppercase {
      text-transform: uppercase;
    }
    p.lowercase {
      text-transform: lowercase;
    }
    p.capitalize {
      text-transform: capitalize;
    }
  </style>

  <p class="uppercase">This is sample text.</p>
  <p class="lowercase">This is sample text.</p>
  <p class="capitalize">This is sample text.</p>

break-spaces

다음 차이점을 제외하면 pre-wrap 과 동일하다.

  • 연속 공백이 줄의 끝에 위치하더라도 공간을 차지한다.
  • 연속 공백의 중간과 끝에서도 자동으로 줄을 바꿀 수 있다.
  • 유지한 연속 공백은 pre-wrap 과 달리 요소 바깥으로 넘치지 않으며, 공간도 차지하므로 요소의 크기(min-content, iconmax-content)에 영향이 미친다.

그림자 효과

text-shadow 속성은 글자에 그림자를 추가한다.

글씨와 그 장식에 적용할 X 및 Y 간격, 흐림 반경, 색상의 조합으로 설정한다.

구문(Syntax)

/* offset-x | offset-y | blur-radius | color */
text-shadow: 3px 3px 5px red;

/* color | offset-x | offset-y | blur-radius */
text-shadow: red 3px 3px 5px;
