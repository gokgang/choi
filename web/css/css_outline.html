윤곽선(outline)

윤곽선은 요소의 바깥에 그려지는 선으로 요소를 눈에 띄게 만들어 준다.

    윤곽선 속성

  • outline-style
  • outline-color
  • outline-width
  • outline-offset

참고 : 윤곽선은 테두리와 다르다! 테두리와 달리 윤곽선은 요소의 테두리 바깥에 그려지며 다른 요소의 윤곽선과 겹칠 수 있다. 또한 윤곽선은 요소의 일부가 아니므로 요소 전체의 너비와 높이는 윤곽선의 영향을 받지 않는다.

    윤곽선 모양

    outline-style 속성은 다음 가운데 하나의 속성을 가질 수 있다.

  • dotted - 점선
  • dashed - 파선
  • solid - 실선
  • double - 이중 실선
  • groove - 고랑 모양
  • ridge - 솟아오름
  • inset - 오목하게
  • outset - 볼록하게
  • none - 없음
  • hidden - 숨김

다양한 윤곽선 모양

  p.dotted{outline-style: dotted;}
  p.dashed{outline-style: dashed;}
  p.solid{outline-style: solid;}
  p.double{outline-style: double;}
  p.groove{outline-style: groove;}
  p.ridge{outline-style: ridge;}
  p.inset{outline-style: inset;}
  p.outset{outline-style: outset;}

윤곽선 너비(outline-width)

outline-width 속성은 다음 값 가운데 하나로 윤곽선의 너비를 설정한다.

  • thin - 가늘게, 1px
  • medium - 보통, 3px
  • thick - 두껍게, 5px
  • px, pt, cm, em 등으로 크기값 설정
  p.thin {
      border: 1px solid black;
      outline-style: solid;
      outline-color: red;
      outline-width: thin;
  }
  p.medium {
      border: 1px solid black;
      outline-style: solid;
      outline-color: red;
      outline-width: medium;
  }
  p.thick {
      border: 1px solid black;
      outline-style: solid;
      outline-color: red;
      outline-width: thick;
  }
  p.w4px {
      border: 1px solid black;
      outline-style: solid;
      outline-color: red;
      outline-width: 4px;
  }

윤곽선 색상

윤곽선의 색상은 outline-color 속성을 사용한다.

  • 색상 이름 - "red"와 같은 색상 이름으로 설정
  • HEX - "#ff0000" 과 같은 16진수 값으로 설정
  • RGB - "rgb(255, 0, 0)과 같은 RGB값으로 설정
  • HSL - "hsl(0, 100%, 50%)와 같은 HSL값으로 설정
  • invert - 색상 반전으로 설정(바탕색에 관계없이 윤곽선이 나타난다.)

다음 예는 색상이 다른 몇 가지 윤곽선을 보여준다. 이러한 요소에는 윤곽선 안에 가늘고 검은 색 테두리가 생긴다. 

  p {
    border: 2px solid black;
    outline-style: solid;
    outline-color: red;
    padding: 10px;
  }

윤곽선 줄임 속성(Outline Shorthand)

    outline 속성은 다음과 같은 개별 윤곽선 속성을 설정하기 위한 줄임 속성이다.
  • outline-width
  • outline-style(필수)
  • outline-color

outline 속성은 위 목록에서 하나 이상의 속성을 설정할 수 있으며, 설정값의 순서는 상관없다.

  p.dashed {outline: dashed;}
  p.dotted {outline: dotted;}
  p.solid {outline: 5px solid yellow;}
  p.thick {outline: thick ridge pink;}

윤곽선 보탬(offset)

outline-offset 속성은 요소의 윤곽선과 가장자리 / 테두리 사이에 공간을 추가한다. 윤곽선과 요소 사이는 빈 공간이다.

다음 예제는 테두리의 가장자리 바깥에 15px의 윤곽선을 설정한다. 

  p.offset {
    margin: 30px;
    border: 1px solid grey;
    outline: 1px solid red;
    outline-offset: 15px;
  }

이 단락에는 테두리 가장자리 15px의 바깥에 윤곽선이 있다.

다음 예제는 요소와 윤곽선 사이에 빈 공간이 있음을 보여준다.

  p.outline {
      margin: 30px;
      background: yellow;
      border: 1px solid grey;
      outline: 1px solid red;
      outline-offset: 15px;
  }

이 단락에는 테두리 가장자리 15px의 바깥에 윤곽선이 있다.

outline-width 바깥선 너비 설정
