CSS 기본

CSS는 웹 문서의 양식을 지정하는 데 사용하는 언어이다.

  • CSS는 Cascading Style Sheets를 의미한다.
  • CSS는 HTML 요소가 화면, 종이 또는 기타 미디어에 표시되는 양식을 선언한다.
  • CSS는 많은 작업을 줄여 주고 한 번에 여러 웹 페이지의 양식을 제어 할 수 있다.
  • 외부 스타일 시트는 CSS 파일을 사용하면 여러 개의 문서가 공유할 수 있다.
  <!DOCTYPE html>
  <html>
  <head>
  <!-- CSS는 style 태그 안에 작성한다. -->
    <style>
    /* body 의 바탕색을 lightblue로 선언한다. */
    body {
        background-color: lightblue;
      }

      /* h1 태그로 매듭을 한 글을 빨간색에 가운데 맞춤을 한다. */
      h1 {
        color: white;
        text-align: center;
      }

      p {
        font-family: verdana;
        font-size: 20px;
      }
    </style>
  </head>
  <body>

    <h1>CSS 예제 </h1>
    <p>이 글은 예제를 위한 글입니다.<p>

  </body>
  </html>
