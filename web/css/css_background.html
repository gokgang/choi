CSS 바탕

CSS 바탕(background) 속성은 바탕 양식을 지정한다.

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • background (줄임 속성)

바탕색

  <style>
    body {
      background-color: lightblue;
    }
  </style>

    색상 값

  • "red" 와 같은 브라우저에서 사용 가능한 색상 이름
  • "#FF0000"과 같은 HEX 값
  • "rgb (255, 0, 0)"과 같은 RGB 값

HTML 요소의 바탕색 지정

  <style>
    h1 {
      background-color: green;
    }
    div {
      background-color: lightblue;
    }
    p {
      background-color: yellow;
    }
  </style>

불투명도(opacity)

opacity 속성은 요소의 불투명도를 지정한다.
0.0 ~ 1.0 사이의 값을 가질 수 있다.
값이 낮을수록 더 투명해 진다.

불투명도 1.0
불투명도 0.6
불투명도 0.3
불투명도 0.0 
  <style>
    div {
      background-color: green;
      opacity: 0.3;
    }
  </style>

참고 : opacity 속성을 사용하여 요소의 바탕에 투명도를 추가하면 모든 하위 요소가 동일한 투명도를 상속한다. 이로 인해 완전히 투명한 요소는 글자를 읽기가 어려울 수 있다.

RGBA를 사용한 투명도

하위 요소에 불투명도를 적용하지 않으려면 RGBA 색상 값을 사용하면 된다.
다음은 바탕색의 불투명도를 설정해도 글자는 영향을 받지 않는다.

불투명도 1.0
불투명도 0.6
불투명도 0.3
불투명도 0.0 
  <style>
    div {
      background-color: rgba(0, 128, 0, 0.3); /* Green background with opacity 30% */
    }
  </style>
