CSS 풀이(Comments)

CSS 풀이는 브라우저에 나타나지 않지만 소스 코드를 풀이하는 데 도움이 된다.

풀이(Comments)

풀이는 주석 또는 설명문이라고도 하며, 코드를 풀이하는데 사용되며, 나중에 코드를 고치려고 할 때 도움이 된다. 풀이는 부라우저가 무시한다.

CSS 풀이는 <style> 요소 안에서 양식을 선언할 때 /* 와 */ 안에 작성한다.

예 : 한 줄 풀이

  /* p 요소의 글자 색 : 빨간색 */
  p {
    color: red;
  }

예 : 문장 뒤의 풀이

  p {
    color: red; /* p 요소의 글자 색 : 빨간색 */
  }

예 : 여러 줄 묶음 풀이

  /* 
    경고문 설정
    p 요소의 바탕색 : 빨간색
    p 요소의 글자생 : 노란색
  */
  p.warning {
    background-color: red;
    color: yellow;
  }

HTML 및 CSS 풀이

HTML 풀이의 <!-- ... --> 와 CSS 풀이 /* ... */ 는 같은 웹 문서에서 사용하게 되므로 적절하게 조화하여 사용해야 한다. 

  <!DOCTYPE html>
  <html>
  <head>
    <style>
      p {
        color: red;   /* 글자색 빨강 */
      }
    </style>
  </head>
  <body>
    <h2>HTML 풀이와 CSS 풀이 </h2>

    <!-- 문단의 글씨는 빨간색이다. -->
    <p>Hello JavaScript!</p>
    <p>이 단락은 CSS로 양식을 지정하였다.</p>
    <p>CSS 풀이는 무시되므로 화면에 나타나지 않는다.</p>
  </body>
  </html>
