안 띄우기

안 띄우기(padding)는 요소의 테두리와 요소의 내용 사이에 띄우기를 지정한다.

이 요소의 안 띄우기는 50px이다.

안 띄우기 속성(padding)

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

속성값

  • length : px, pt, cm 등으로 지정
  • % : 포함하는 상위 요소 너비의 백분율로 지정
  • inherit : 포함하는 상위 요소에서 상속

    • 참고 : 음수 값 불가

  <style>
    div {
      border: 1px solid grey;
      padding-top: 50px;
      padding-right: 30px;
      padding-bottom: 50px;
      padding-left: 80px;
    }
  </style>

줄임 속성(padding)

  안 띄우기의 줄임 속성 padding 으로 4개의 속성값을 한꺼번에 모두 설정할 수 있다. top right bottom left 순으로 시계 회전 방향으로 나열하며, 속성값이 일부 생략되면 서로 마주 보는 쪽의 값이 적용된다. 3개의 속성값만 있으면 마지막의 left는 right와 같은 값이 적용되고, 2개의 속성값이 있으면 top 과 bottom, right 와 left 가 같은 값이 적용되며, 1개의 속성값이 있으면 모든 안 띄우기가 같은 값이 적용된다. 

  <style>
    div {
      border: 1px solid grey;
      padding: 50px 30px 50px 80px;
    }
  </style>

요소 너비(width) 와 안 띄우기(padding)

너비(width)가 지정된 요소에 안 띄우기(padding)를 추가하면 요소의 너비에 안 띄우기 값이 추가되므로 전체 너비가 늘어난다.

다음과 같이 설정한 경우 : 

  div {
    width: 300px;
    padding: 25px;
  }

실제 너비는 width: 300px + padding-left: 25px + padding-rignt: 25px = width: 350px 가 된다.

안 띄우기에 관계없이 너비를 300px로 유지하려면 box-sizing 속성을 사용하여 요소의 너비를 유지할 수 있다. 

  div {
    width: 300px;
    padding: 25px;
    box-sizing: border-box;
  }

이렇게 하면 사용 가능한 공간이 줄어 든다.
