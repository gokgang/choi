CSS 테두리 모양

border-style: 테두리 모양 지정

  • dotted - 점선
  • dashed - 파선
  • solid - 연속선
  • double - 이중선
  • groove - 3D groobe, 색상에 따라 다른 효과
  • ridge - 3D 능선, 색상에 따라 다른 효과
  • inset - 3D 삽입, 색상에 따라 다른 효과
  • outset - 3D 시작 테두리, 색상에 따라 다른 효과
  • none - 테두리 없음
  • hidden - 테두리 숨김

테두리의 모양

border-style: 아래 위 왼쪽 오른쪽 모든 테두리 모양

p.dotted {border-style: dotted;}

p.dashed {border-style: dashed;}

p.solid {border-style: solid;}

p.double {border-style: double;}

p.groove {border-style: groove;}

p.ridge {border-style: ridge;}

p.inset {border-style: inset;}

p.outset {border-style: outset;}

p.none {border-style: none;}

p.hidden {border-style: hidden;}

p.mix {border-style: dotted dashed solid double;}

모든 태두리 속성

속성설명
border 모든 테두리 속성을 하나의 선언으로 설정
border-bottom 모든 아래쪽 테두리 속성을 하나의 선언으로 설정
border-bottom-color 아래쪽 테두리 색상 설정
border-bottom-style 아래쪽 테두리 모양 설정
border-bottom-width 아래쪽 테두리 두께 설정
boder-color 모든 테두리 색상 설정
border-left 모든 왼쪽 테두리 속성 설정
border-left-color 왼쪽 테두리 색깔 설정
border-left-style 왼쪽 테두리 모양 설정
border-left-width 왼쪽 테두리 두께 설정
border-radius 둥근 모서리의 반지름 설정
border-right 오른쪽 테두리 속성을 모두 설정
border-right-color 오른쪽 테두리의 색깔 설정
border-right-style 오른쪽 테두리 모양 설정
border-right-width 오른쪽 테두리 두께 설정
border-style 모든 테두리의 모양 설정
border-top 위쪽 테두리 속성을 모두 설정
border-top-color 위쪽 테두리 색상 설정
border-top-style 위쪽 테두리 모양 설정
border-top-width 위쪽 테두리의 너비 설정
border-width 모든 테두리의 너비 설정

border-style 속성이 지정되어야 다른 속성의 효과가 있다. 색상 설정을 하지 않으면 기본값인 border-color: black 으로 표시된다.
