바탕 그림

background-image 속성은 요소의 바탕에 사용할 그림 지정한다.
기본적으로 그림은 전체를 덮도록 반복된다.
Background Image 바탕 그림 반복 바탕 그림 위치 바탕 그림 고정 바탕 양식 줄임 속성

Background Image

바탕 그림은 특정 요소에 설정할 수도 있다. 

  <style>
    div {
      background-image: url("paper.gif");
    }
  </style>

Hello World!

이 DIV 요소에는 바탕 그림이 있다!

잘못된 조합

다음은 바탕에 사용된 그림 때문에 글을 읽기 어렵다. 

  <style>
    div {
      background-image: url("bgdesert.jpg");
    }
  </style>

Hello World!

이 DIV 요소는 바탕 그림 때문에 글을 읽기가 어렵다!

바탕 그림 반복

기본적으로 이 background-image 속성은 이미지를 가로 및 세로로 반복한다.
일부 이미지는 가로 또는 세로로 만 반복되어야 한다. 그렇지 않으면 다음과 같이 이상하게 보인다. 

  <style>
    div {
      background-image: url("gradient_bg.png");
    }
  </style>

Hello World!

이상한 바탕 그림 ...

위의 바탕 그림이 가로 ( background-repeat: repeat-x;) 로만 반복되면 배경이 더 좋아 보인다. 

  <style>
    div {
      background-image: url("gradient_bg.png");
      background-repeat: repeat-x;
    }
  </style>

Hello World!

여기에서는 바탕 그림이 가로로만 반복된다!

참고 : 이미지를 세로로 반복하려면 background-repeat: repeat-y;

반복 없음

바탕 그림을 한 번만 표시하려면 background-repeat 속성을 no-repeat 로 지정한다 . 

  <style>
    div {
      background-image: url("img_tree.png");
      background-repeat: no-repeat;
    }
  </style>

Hello World!

바탕 그림이 한 번만 표시되지만 그림이 글자와 겹쳐서 읽어보기가 어렵다.

바탕 그림 위치

background-position 속성은 바탕 그림의 위치를 ​​지정할 수 있다.

다음은 오른쪽 위 모서리에 바탕 그림을 배치한다. 

  <style>
    div {
      background-image: url("img_tree.png");
      background-repeat: no-repeat;
      background-position: right top;
    }
  </style>

Hello World!

바탕 그림은 한 번만 표시되고 글에서 멀리 떨어져 있다.
이 예에서는 오른쪽에 여백도 추가하여 바탕 그림이 글자를 방해하지 않는다.

바탕 그림 고정

background-attachment 속성은 바탕 그림을 화면을 따라 움직일 지 고정할지 여부를 지정한다.

바탕 그림 고정 

  <style>
    body {
      background-image: url("img_tree.png");
      background-repeat: no-repeat;
      background-position: right top;
      background-attachment: fixed;
    }
  </style>

바탕 그림 글과 함께 따라서 움직임 

  <style>
    body {
      background-image: url("img_tree.png");
      background-repeat: no-repeat;
      background-position: right top;
      background-attachment: scroll;
    }
  </style>

바탕 양식 줄임 속성

코드를 줄이려면 하나의 속성에 모든 바탕 양식을 지정할 수 있다. 

  <style>
    body {
      background: #ffffff url("img_tree.png"), no-repeat right top;
    }
  </style>

    줄임 속성의 속성값 순서

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position

순서에 있는 속성 값 중 하나가 누락되었는지 여부는 중요하지 않다. 위의 예에서 background-attachment 속성은 값이 없기 때문에 사용하지 않았다.
