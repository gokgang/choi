양식(Style Sheets) 넣기

웹 브라우저는 CSS의 정보에 따라 HTML 문서의 양식을 지정한다.

양식 넣기

  • 문서 외부 CSS 파일 연결
  • 문서의 <head> 부분에 <style> 선언
  • HTML 요소의 style 속성에 설정

외부(External)의 양식(Style Sheets)

외부 양식를 사용하면 하나의 파일만 변경하여 전체 웹 문서의 양식을 변경할 수 있다.
각 HTML 문서에는 <head> 안에 <link> 요소를 사용하여 외부 양식 파일에 대한 참조를 지정한다.
외부 양식 파일 확장자는 *.css 이다.
외부 양식 파일에는 HTML 요소의 매듭(Tag)이 없어야 한다. 

  <!DOCTYPE html>
  <html>
  <head>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    ...
  </body>
  </html>
file name : "style.css"
  body {
    background-color: lightblue;
  }

  h1 {
    color: navy;
    margin-left: 36px;
  }

단위는 항상 값에 붙여서 사용해야 한다.

내부 양식(Internal Style) 넣기

하나의 HTML 문서 만이 고유 양식으로 만들 경우 내부 양식을 사용한다. 내부 양식은 <head> 부분에 양식을 정의한 <style> 요소를 배치한다. 

  <!DOCTYPE html>
  <html>
  <head>
    <style>
      body {
        background-color: linen;
      }
      h1 {
        color: maroon;
        margin-left: 36px;
      }
    </style>
  </head>
  <body>
    ...
  </body>
  </html>

요소 안의 양식(Inline Style) 지정

단일 요소에 고유 양식을 지정하려면 요소에 style 속성을 설정한다.
style 속성은 모든 CSS 속성을 지정할 수 있다. 

  <h1 style="color: blue; text-align: center;">요소 안의 양식 지정</h1>
  <p style="color: darkcyan;">특정 요소에 <em style="color: red;">style</em> 속성을 설정한다.</p>

요소 안의 양식 지정

특정 요소에 style 속성을 설정한다.

요소 안에 양식 지정을 많이 사용하면 문서가 매우 혼잡해짐으로 특정 요소 만 고유 양식을 지정하는 경우에 사용해야 한다.

양식의 중복

같은 요소에 양식이 중복하여 정의된 경우 맨 마지막에 정의된 양식이 사용된다.
외부의 양식 파일과 문서 내부에 지정된 양식, 그리고 요소의 style 속성에 양식을 모두 다르게 지정한 경우에 해당 요소에 가장 가까운 곳에 있는 양식이 계단적( Cascading )으로 적용된다.

우선 순위

  1. 요소 안에 지정한 양식( Inline Style )
  2. <head> 부의 양식( Internal Style )과 외부 파일의 양식( External Style ) 중 나중의 양식
  3. 웹 브라우저 기본값
  <!-- 외부 파일 : "mystyle.css" -->
  h1 {
    color: navy;
  }

  <head>
    <link rel="stylesheet" href="mystyle.css">
    <!-- 외부 양식 파일을 연결 후 내부 양식 선언 -->
    <style>
      h1 {
        color: orange;
      }
    </style>
  </head>

  <head>
    <style>
      h1 {
        color: orange;
      }
    </style>
    <!-- 외부 양식 파일을 내부 양식 선언 보다 나중에 연결 -->
    <link rel="stylesheet" href="mystyle.css">
  </head>

  <head>
    <link rel="stylesheet" href="mystyle.css">
    <style>
      h1 {
        color: orange;
      }
    </style>
  </head>
  <body>
    <style>
    /* <body>에서  Style 선언 */
      h1 {
        color: magenta;
      }
    </style>
    <h1><body>에서 style 선언</h1>
  </body>
