상자 모형(Box Model)

모든 HTML 요소는 상자로 간주할 수 있다. CSS에서 상자 모형(box model)이라는 용어는 웹 페이지의 설계와 배치를 할 때 사용한다.
CSS 상자 모형은 기본적으로 모든 HTML 요소를 감싸는 상자이다. 상자 모형은 바깥 띄우기, 테두리, 안 띄우기 및 실제 내용으로 구성된다. 내용이란 글, 그림, 동영상 등의 자료를 말한다. 아래에는 상자 모형의 구조를 보여준다.

바깥 띄우기(margin)
테두리(border)
안 띄우기(padding)

내용(content)

 
 
 
  • 내용(content) - 글자와 그림, 동영상 등의 자료
  • 안 띄우기(padding) - 내용과 테두리 사이의 간격, 배경은 내용과 같다.
  • 테두리(border) - 안 띄우기와 내용을 감싸는 테두리, 기본값은 선이 없다.
  • 바깥 띄우기(margin) - 테두리 바깥의 다른 요소와의 간격이다. 색상은 상위 요소의 값을 따르며, 기본값은 각 웹 브라우저 마다 다르다.

※ HTML 요소는 모두 상자 모형이므로 요소의 테두리를 설정하고, 테두리 안밖의 띄우기를 설정할 수 있다.

  div {
    background-color: lightgrey;
    width: 300px;
    border: 15px solid green;
    padding: 50px;
    margin: 20px;
  }

HTML 요소의 너비와 높이

CSS로 너비 및 높이를 설정할 때 내용 영역의 너비와 높이의 설정만해도 되지만, 요소 전체 크기를 맞추려면 안 띄우기, 바깥 띄우기 및 테두리도 설정해 주어야 한다.

이 <div> 요소의 총 너비는 350px이다.

  div {
    width: 320px;
    padding: 10px;
    border: 5px solid gray;
    margin: 0;
  }

※ 위 예제의 계산은 다음과 같다.

    320px(width) 
    + 20px(padding-left: 10px + padding-right: 10px)
    + 10px(border-lft: 5px + border-right: 5px) 
    + 0px(margin-left: 0px + margin-right:0px)
    = 350px

요소의 총 너비 계산

요소 총 너비 = 너비 + 왼쪽 안 띄우기 + 오른쪽 안 띄우기 + 왼쪽 테두리 + 오른쪽 테두리 + 왼쪽 바깥 띄우기 + 오른쪽 바깥 띄우기

요소의 총 높이 계산

요소 총 높이 = 높이 + 위쪽 안 띄우기 + 아래쪽 안 띄우기 + 위쪽 테두리 + 아래쪽 테두리 + 위쪽 바깥 띄우기 + 아래쪽 바깥 띄우기
