높이(height) 와 너비(width)

요소의 높이는 height , 너비는 width 를 사용하여 설정한다.
요소의 최대 너비는 max-width 로 지정한다. 

  <style>
    div {
      height: 200px;
      width: 100%;
      max-width: 600px;
      border: 1px solid skyblue;
    }
  </style>

height 및 width 속성에는 padding, border, margin 등이 포함되지 않는다.
