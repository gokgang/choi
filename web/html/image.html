그림 넣기

구문 너비와 높이 Image Link 그림 배치 그림 형식 이미지 맵

HTML <img> 요소는 웹 문서에 그림을 포함하는 데 사용한다.

그림은 웹 문서에 삽입되지는 않는다. 그림은 웹 문서에 참조로 연결된다.
<img> 요소는 참조 그림에 대한 유지 공간을 만든다.
<img> 요소는 속성만 있으며, 끝 매듭이 없는 빈 요소이다.

<img> 요소의 다음 두 개의 속성은 필수이다.

  • src - 그림의 파일 경로 지정
  • alt - 그림을 보여줄 수 없을 때 대체할 글

구문

<img src="url" alt="alternatetext">

참고 : 웹 페이지가 로드 될 때 웹 서버에서 이미지를 가져와 페이지에 삽입하는 것은 브라우저이다. 이미지가 실제로 웹 페이지와 관련하여 url이 일치하지 않으면 방문자의 웹 브라우저에 끊어진 링크 아이콘이 표시되고, alt 에 지정한 대체글이 표시된다. 

  <img src="img_flower.jpg" alt="진달래 꽃">
  <img src="image/img_flower.jpg" alt="진달래 꽃">

첫 번째 <img> 요소는 경로가 맞지 않아서 링크 아이콘과 대체글이 나타난다.

브라우저가 이미지를 찾을 수 없는 경우 alt 속성 값이 표시된다. 뿐만 아니라 시각 장애가 있거나 학습 장애가 있는 사람들에게 화면 판독기 또는 스크린 리더로 하여금 HTML 코드를 읽고 사용자가 문서의 내용을 들을 수 있도록 한다.

너비( width )와 높이( height )

width 와 height 속성 설정 
  <img src="image/img_flower.jpg" alt="진달래 꽃" width="110" height="166">
style 속성 width 와 height 설정 
  <img src="image/img_flower.jpg" alt="야생화" style="width: 150px; height: 100px;">

width와 height 속성으로 크기를 설정한 경우에 CSS에서 그림의 크기를 설정하면 바뀌게 되므로 style 속성을 사용하면 이것을 막을 수 있다. 

  <!DOCTYPE html>
  <html>
  <head>
    <style>
      img {
        width: 100%;
      }
    </style>
  </head>
  <body>
    <img src="img_firstlove.jpg" alt="야생화" width="150" height="100">
    <img src="img_firstlove.jpg" alt="야생화" style="width:150px;height:100px;">
  </body>
  </html>

Image Link

  <a href="default.html">
    <img src="image/img_home.png" alt="Go Home image" style="width: 120px;width: 40px;">
  </a>

그림을 클릭하면 HTML 홈으로 이동

이미지 맵

HTML 이미지 맵을 사용하면 이미지에서 클릭 가능한 영역을 만들 수 있다.

HTML <map> 요소는 이미지 맵을 정의한다. <area> 요소 로 정의한 그림은 클릭 가능한 영역이 있는 그림이 된다.

이미지 맵을 만들려면 문서에 <img> 매듭을 사용하여 그림을 삽입할 때 usemap 속성을 추가하면 된다.
usemap 속성의 값은 hash (#)로 붙인 이미지 맵의 이름이다.

그림 삽입 img

<img src="img_map.jpg" alt="image map" usemap="#linkmap">

지도 만들기 map

<map name="linkmap"></map>

<map> 요소의 name 속성을 <img> 요소의 usemap 속성과 같은 값으로 설정한다.

영역 지정 area

<area shape="rect | circle | poly | default" coords="coords value" href="url" alt="image map">
    클릭 가능한 영역의 모양
  • rect ― 사각형
  • circle ― 원형
  • poly ― 다각형
  • default ― 그림 전체

사각형 영역

shape="rect" 는 coods value가 2쌍의 x 축 좌표와 y 축 좌표로 구성되며, 앞에는 왼쪽 위 좌표이고 이어서 오른쪽 아래의 좌표이다. 

  <img src="./image/img_menu.png" alt="image menu" usemap="#linkmap">
  <map name="linkmap">
      <area shape="rect" coords="0, 0, 200, 54" href="/" alt="image Home">
      <area shape="rect" coords="201, 0, 400, 54" href="default.html" alt="image 방명록">
      <area shape="rect" coords="401, 0, 600, 54" href="default.html" alt="image 게시판">
      <area shape="rect" coords="601, 0, 800, 54" href="default.html" alt="image 알림">
      <area shape="rect" coords="801, 0, 1000, 54" href="default.html" alt="image 축하">
  /map>

원형 영역circle

맵으로 사용할 원의 중심좌표(x, y)와 원의 반지름( px ) 

  <img src="img_map.jpg" alt="image map" usemap="#linkcircle">
  <map name="linkcircle">
      <area shape="circle" coords="x-axis, y-axis, radius" href="url" alt="image circle">
  /map>

  <img src="image/img_firstlove.jpg" alt="야생화" usemap="#linkcircle">
  <map name="linkcircle">
    <area shape="circle" coords="420, 220, 160" href="image/img_firstlove.jpg" target="_blank" 
    alt="꽃말 첫사랑" style="width: 800px;height: 500px;">
  </map>

꽃 주위에 원형 링크 설정

다각형 영역poly

맵으로 사용할 모든 꼭지점의 좌표(x, y)를 시계회전 방향으로 나열하면 각각의 꼭지점이 연결된 다양한 모양의 영역을 만들수 있다. 

  <img src="img_map.jpg" alt="image map" usemap="#linkpoly">
  <map name="linkpoly">
      <area shape="poly" coords="x1, y1, x2, y2, x3, y3, x4, y4, x5, y5, " href="url" alt="image poly">
  /map>

  <img src="image/img_firstlove.jpg" alt="야생화" usemap="#linkpoly">
  <map name="linkpoly">
    <area shape="poly" 
    coords="460,20, 435,172, 580,185, 390,261, 164,190, 300,190, 250,110, 338,170, 390,96" 
    href="image/img_firstlove.jpg" target="_blank" alt="꽃말 첫사랑" style="width: 800px;height: 500px;">
  </map>

꽃잎에만 다각형의 링크 설정

이미지 맵과 JavaScript 의 조합

  <img src="image/img_flower.jpg" alt="진달래" usemap="#mapflower" style="width: 276px;height: 416px;">
  <map name="mapflower">

  </map>

배경 이미지

거의 모든 HTML 요소에 대해 배경 이미지를 지정할 수 있다.
HTML 요소에 배경 이미지를 사용하려면 HTML style 속성과 CSS background-image 속성을 사용하여 지정한다.

HTML div 요소에 배경 이미지 추가 : 

  <div style="background-image: url('./image/img_bg.png');">
background-image

문서 배경 이미지

  <head>
    <style>
      body {
        background-image: url('image/img_bg.png');
      }
    </style>
  </head>

배경 이미지 반복

배경 이미지가 요소보다 작으면 이미지가 요소의 끝에 도달할 때까지 가로 및 세로로 반복된다.
배경 이미지가 반복되지 않도록하려면 background-repeat 속성을 no-repeat 로 설정한다. 

  <head>
    <style>
      body {
        background-image: url('image/img_bg.png);
        background-repeat: no-repeat;
      }
    </style>
  </head>

전체 그림 배경

배경 그림이 전체 요소를 덮도록하려면 background-size 속성을 다음과 같이 cover 설정할 수 있다.
또한 전체 요소가 항상 덮여 있는지 확인하려면 background-attachment 속성을 fixed 로 설정하면 배경 이미지가 늘어나지 않고 원래 비율을 유지하면서 전체 요소를 덮게 된다. 

  <head>
    <style>
      body {
        background-image: url('image/img_flower.jpg');
        background-repeat: no-repeat;
        background-attachment: fixed;
        background-size: cover;
      }
    </style>
  </head>

배경 그림 늘이기

배경 그림이 전체 요소에 맞게 늘어나도록 하려면 background-size 속성을 100% 로 설정한다 .
background-size: 100% 와 같이 설정하면 가로 세로의 비율을 유지하면서 늘어나고, background-size: 100% 100%와 같이 설정하면 가로 세로가 모두 화면 크기로 늘어나므로 원래 그림의 가로 세로 비율과는 다르게 늘어난다. 

  <head>
    <style>
      body {
        background-image: url('image/img_flower.jpg');
        background-repeat: no-repeat;
        background-attachment: fixed;
        background-size: 100% 100%;
      }
    </style>
  </head>

<picture> 요소

화면 크기에 따라 다른 그림을 보여준다. 그리고 지정한 그림을 보여줄 수 없는 경우 대체 그림을 보여준다. 

  <picture>
    <source media="(min-width:1280px)" srcset="image/img_flower.jpg">
    <source media="(min-width:800px)" srcset="image/img_firstlove.jpg">
    <img src="img_flower.png" alt="flower" style="width: auto;">
  </picture>

화면의 크기를 늘이거나 줄이면 다른 그림이 보여진다.

그림(picture) 요소를 사용하는 경우 작은 화면의 장치에 큰 이미지 파일을 로드할 필요가 없다. 브라우저는 <source> 속성 값이 일치하는 첫 번째 요소를 사용하고 다음 요소를 무시한다.
일부 브라우저 또는 장치는 모든 이미지 형식을 지원하지 않을 수 있다. <picture>요소를 사용하면 모든 형식의 이미지를 추가할 수 있으며 브라우저는 인식하는 첫 번째 형식을 사용하고 다음 요소는 무시한다.
