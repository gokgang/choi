뜻있는 요소(Semantic Elements)

뜻있는 매듭(Semantic Tags)

뜻있는 요소(semantic elements)는 브라우저와 개발자 모두에게 그 뜻을 알기 쉽다.
뜻없는 요소(non-semantic Elements) : <div> 와 <span> 요소는 아무런 뜻이 없다.
뜻있는 요소(semantic elements) : <form>, <table>, <article> 등은 그 내용의 뜻을 알기 쉽다.

많은 웹 사이트는 탐색(<div id = "nav">), 머리글(<div class = "header">) 및 바닥글(<div id = "footer">) 을 나타내는 HTML 코드가 포함되어 있다.

HTML은 웹 페이지의 다른 부분을 정의할 수 있는 몇 가지 뜻있는 요소가 있다.

  • <article>
  • <aside>
  • <details>
  • <figcaption>
  • <figure>
  • <footer>
  • <header>
  • <main >
  • <mark >
  • <nav>
  • <section>
  • <summary >
  • <time>

구역 요소 <section>

section 요소는 문서의 구역을 지정한다. 구역은 제목이 있는 주제별 내용의 모음이다. 

  <section>
    <h1>WWF</h1>
    <p>
      세계 자연보호 기금(WWF)은 이전에 세계 야생생물 기금(World Wildlife Fund)이라고 
      불렸던 환경의 보존, 연구, 복원과 관련된 문제에 관한 국제기구이다. WWF는 1961년에 
      설립되었다.
    </p>
  </section>
  <section>
    <h1>WWF's 팬더 기호</h1>
    <p>
      팬더는 WWF의 상징이 되었다. WWF의 유명한 팬더 기호는 WWF가 설립된 같은 해에 
      베이징 동물원에서 런던 동물원으로 옮겨진 치치라는 팬더에서 유래되었다.
    </p>
  </section>

이야기 요소 <article>

<article> 요소는 독립적인 글을 지정한다. 이야기는 그 자체로 뜻이 있고 다른 웹 문서와 별도의 글이다.

<article> 요소의 사용 예:

  • 토론(Forum) 게시물
  • 사랑방(Blog) 게시물
  • 신문(Newspaper) 기사

<article> 요소는 독립적인 내용을 지정하며, <section> 요소는 문서의 구역을 지정한다.
이 두 개의 요소는 서로 다른 뜻의 요소이므로 서로의 요소를 포함할 수 없다.

<header> 요소

<header> 요소는 소개글이나 탐색 링크들을 싣는 수레(container)이다.

<header> 요소에 포함하는 내용

  • 하나 이상의 제목 요소(<h1> ~ <h6>)
  • 상징 로고 또는 상징 아이콘
  • 개발자 정보

참고 : <header> 요소 안에 여러가지 요소가 있을 수 있지만 <footer>, <address> 그리고 또 다시 <header> 요소를 배치할 수 없다. 

  <article>
    <header>
      <h1>WWF</h1>
      <p>
        WWF의 임무
      </p>
    </header>
    <p>
      세계 야생생물 기금의 임무는 지구의 자연환경 악화를 막는 것이다.
      그리고 인간이 자연과 조화롭게 사는 미래를 건설한다.
    </p>
  </article>

<footer> 요소

<footer> 요소는 문서나 구역의 바닥글을 정의한다.

<footer> 요소에 포함하는 내용

  • 저자 정보
  • 저작권 정보
  • 연락 정보
  • 사이트 구성
  • 맨 위로 바로가기
  • 참고 문헌

<footer> 요소에도 여러가지 요소가 있을 수 있다. 

  <footer>
    <p>개발자 : 박문수</p>
    <p><a href="mailto:park@example.com">park@example.com</a></p>
  </footer>

<nav> 요소

<nav> 요소는 탐색 링크를 지정한다.
문서의 모든 링크가 <nav>요소 안에 있어야 하는 것은 아니다 . 이 <nav> 요소는 주요 탐색 링크만 지정한다.
화면 판독기 사용자를 위하여 브라우저는 이 요소를 사용하여 초기 설명을 결정할 수 있다.

탐색 링크 설정

  <nav>
    <a href="/html/default.html">HTML</a> |
    <a href="/css/default.html">CSS</a> |
    <a href="/script/default.html">JavaScript</a>
  </nav>
HTML | CSS | JavaScript

<figure>, <figcaption> 요소

<figure> 요소는 그림, 도표, 사진, 코드 목록과 같은 요소를 포함하고, <figcaption> 요소는 <figure> 요소의 처음 또는 마지막에 배치하여 <figure> 요소에 대한 캡션을 지정한다.
<img> 요소는 실제 이미지 / 그림을 지정한다. 

  <figure>
    <img src="image/img_flower.gif" alt="야생화">
    <figcaption>Fig1. - 들에 핀 야생화.<f/igcaption>
  </figure>
