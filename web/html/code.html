code 요소

키보드 입력

<kbd>요소는 키보드 입력을 정의한다. 입력 내용은 브라우저의 기본 고정 폭 글꼴로 표시된다. 

  <p>키보드 입력 : <kbd>Ctrl + S</kbd></p>

키보드 입력 : Ctrl + S

프로그램 출력

<samp>요소는 컴퓨터 프로그램의 샘플 출력을 정의한다. 내부 내용은 브라우저의 기본 고정 폭 글꼴로 표시된다. 

  <p>Message from my computer:</p>
  <p><samp>File not found.<br>Press F1 to continue</samp></p>

Message from my computer:

File not found.
Press F1 to continue

컴퓨터 코드 용 HTML <code>

<code>요소는 컴퓨터 코드를 정의하는 데 사용된다. 내부 내용은 브라우저의 기본 고정 폭 글꼴로 표시된다. 

  <code>
    x = 5;
    y = 6;
    z = x + y;
  </code>
x = 5; y = 6; z = x + y;

<code>요소는 추가 공백과 줄 바꿈을 보존하지 않는다. 이 문제를 해결하려면 <pre>요소 안에 <code> 요소를 포함하면 된다. 

  <pre>
  <code>
    x = 5;
    y = 6;
    z = x + y;
  </code>
  </pre>


  x = 5;
  y = 6;
  z = x + y;

변수 용 HTML <var>

HTML <var>요소는 프로그래밍 또는 수학 표현식에서 변수를 정의하는 데 사용된다. 일반적으로 기울임 꼴로 표시된다. 

  <p>삼각형의 면적 : 1 / 2 × <var>b </var> ×  <var>h </var>,  <var>b </var> 는 밑변,   <var>h </var>는 높이이다.</p>

삼각형의 면적 : 1 / 2 × b × h , b 는 밑변, 　h 는 높이이다.
