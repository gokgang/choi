겉 모양(Layout) 요소

웹 문서의 겉 모양을 만드는 방법은 CSS float, framework, flexbox, grid 등의 속성을 사용하는 여러가지가 있다.

CSS Frameworks

웹 문서의 겉 모양을 빠르게 만들고 싶으면 이미 만들진 W3.CSS 또는 Bootstrap 과 같은 CSS 프레임 워크를 무료로 사용할 수 있다 .

  • <header> - 문서 또는 구역의 머리부분을 정의
  • <nav> - 탐색 Link 정의
  • <section> - 문서의 구역 정의

CSS Flexbox Layout

flexbox를 사용하면 문서의 겉 모양을 사용자의 화면 크기에 맞출 수 있다. 

  <!DOCTYPE html>
  <html lang="ko">
  <head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS flexbox</title>
      <style>
        * {box-sizing: border-box;}
        body {font-family: 'Times New Roman', Times, serif;}
        /* header 양식 지정 */
        header {
          background-color: #def;
          padding: 30px;
          text-align: center;
          font-size: 35px;
          color: darkblue;
        }
        /* Container for flexboxes */
        section {
          display: -webkit-flex;
          display: flex;
        }
        /* 탐색 메뉴 */
        nav {
          -webkit-flex: 1;
          -ms-flex: 1;
          flex: 1;
          background-color: #ccf;
          padding: 20px;
        }
        /* 내부 탐색 목록 */
        nav ul {
          list-style-type: none;
          padding: 0;
        }
        article {
          -webkit-flex: 3;
          -ms-flex: 3;
          flex: 3;
          background-color: #eee;
          padding: 10px; 
        }
        /* footer 양식 */
        footer {
          background-color: #ffc;
          padding: 16px;
          text-align: center;
          color: navy;
        }
        /* 반응형 겉 모양 - 600px 보다 작은 화면은 두 개의 가로 열을 아래 위로 배치 */
        @media (max-width: 600px) {
          section {
            -webkit-flex-direction: column;
            flex-direction: column;
          }
        }
      </style>
  </head>
  <body>
  <header>
    <h2>대한민국 주요 도시</h2>
  </header>

Grid Layout

CSS Grid Layout Module은 행과 열이 있는 그리드 기반 레이아웃 시스템을 제공하므로 float 및 위치 지정을 사용하지 않고도 웹 문서를 더 쉽게 설계할 수 있다.

모든 HTML 요소의 box-sizing 속성을 border-box 로 설정하면 패딩과 테두리가 요소의 전체 너비와 높이에 포함된다.

* {box-sizing: border-box;}
25%
75%

  <style>
    .menu {
      width: 25%;
      float: left;  /* 외쪽 맞춤으로  지정 */
    }
    .main {
      width: 75%;
      float: left;  /* 외쪽 맞춤으로  지정 */
    }
  </style>

12 개의 열이있는 반응형 그리드 뷰

Flexbox Layout

Flexible Box Layout Module을 사용하면 float 또는 position 지정을 사용하지 않고도 반응형 겉모양을 만들 수 있다.

크기가 달라지는(Flexible) 수레(Container) 지정

크기가 달라지는 겉 모양은 표시( display ) 속성이 flex 로 설정된 상위 요소가 있어야 한다.
display 속성이 flex 로 설정된 요소에 담겨진 요소는 display ) 속성을 내려 받는다. 

  <style>
    .flex-container {
      display: flex;
      background-color: dodgerblue;
    }
    .flex-container > div {
      background-color:  #f1f1f1;
      margin: 10px;
      padding: 20px;
      font-size: 30px;
      text-align: center;
    }
  </style>
