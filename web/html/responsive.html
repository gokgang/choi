반응형 웹 문서

반응형 웹 문서는 HTML 및 CSS를 사용하여 웹 문서의 크기를 자동으로 조정하고, 숨기거나, 늘이고 줄여서 데스크톱, 태블릿 및 휴대폰 등 모든 장치에서 보기 좋게 만드는 것이다.

화면 크기 설정

반응형 웹 문서를 만들려면 모든 웹 문서에 다음과 같은 <meta> 요소를 추가해야 한다.
viewport 설정은 브라우저에게 문서의 크기 조정을 위한 지침을 알려준다.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

반응형 그림

CSS width 속성을 100 %로 설정하면 그림이 브라우저 창의 크기에 맞도록 조정된다. 

    <img src="image/img_firstlove.jpg" style="width: 100%;">

최대 너비 지정

max-width 속성을 지정하면 원래 크기 보다 늘어나지 못하도록 할 수 있다. 

    <img src="image/img_firstlove.jpg" style="max-width: 100%; height: auto;">

브라우저 너비에 따라 다른 그림 표시

  <picture>
    <source srcset="./image/img_flower.gif" media="(max-width: 600px)">
    <source srcset="./image/img_flower.jpg" media="(max-width: 1500px)">
    <source srcset="./image/img_firstlove.jpg">
    <img src="./image/img_firstlove.jpg" alt="첫사랑">
  </picture>

반응형 글자 크기

글자 크기는 “화면 너비” 를 의미하는 “vw” 단위로 설정할 수 있다.
글자 크기를 “vw” 단위로 설정하면 글자 크기가 브라우저 창의 크기를 따라 조정된다. 

  <h1 style="font-size: 10vw;">Hello World!</h1>

Hello World!

브라우저 창의 크기를 조정하면 글자 크기가 바뀐다.

Viewport는 브라우저 창의 크기이다. 1vw = Viewport 너비의 1 %. Viewport 너비가 50cm이면 1vw는 0.5cm이다.

Media Query

글과 그림의 크기를 조정하는 것 외에도 Media Query를 사용하여 브라우저 크기에 따라 완전히 다른 모양의 문서를 만들 수 있다. 

  <style>
    * {
      box-sizing: border-box;
    }
    .left, .right {
      float: left;
      width: 20%; /* 구역의 기본 너비 */
    }
    .main {
      float: left;
      width: 60%; /* 본문 기본 너비 */
    }
    /* 화면 너비 800px 보다 작으면 각 부분의 너비를 100%로 바꾸기 */
    @media screen and (max-width: 800px) {
      .left, .main, .right {
        width: 100%;
      }
    }
  </style>

  <header style="background-color: aquamarine; ">
    Header
  </header>
  <section>
    <div class="left">
      <p>Left menu</p>
    </div
    <article class="main">
      <p>Main contents</p>
    </article
    <div class="right">
      <p>Right contents</p>
    </div
  </section>
  <footer style="background-color: lightslategray; ">
    Footer
  </footer>

W3.CSS는 기본적으로 데스크톱, 태블릿 및 모바일을 지원하는 최신 CSS Framework 로 무료이며 사용하기도 쉽다.
W3.CSS는 고품질 Bootstrap 의 대안으로 만들어 졌으며, jQuery 또는 JavaScript library 와도 독립적으로 만들어 졌다.

대한민국 주요 도시

서울

대한민국의 수도인 서울을 지방자치단체인 특별시로 부르는 명칭이다. 한반도 중앙에 있으며, 한강을 사이에 두고 남북으로 펼쳐져 있다. 북쪽 끝은 도봉구 도봉동, 동쪽 끝은 강동구 상일동, 남쪽 끝은 서초구 원지동, 서쪽 끝은 강서구 오곡동이다. 시청은 중구 을지로1가(태평로1가 31)에 있다.

대구

북쪽으로 경상북도 칠곡군·군위군, 동쪽으로 경상북도 경산시·청도군, 남쪽으로 경상남도 창녕군, 서쪽으로 경상북도 고령군·성주군과 접한다. 영남지방의 중앙에서 서쪽으로 치우친 대구분지의 중앙부, 낙동강(洛東江)과 금호강(琴湖江)의 합류지점 동쪽 금호강 연안에 있다.

부산

대한민국 제2의 도시이자, 제1의 무역항이다. 동쪽과 남쪽은 바다에 면하고, 서쪽은 김해시 장유동과 창원시 진해구, 북쪽은 양산시 물금읍과 김해시 대동면, 동쪽은 울산광역시 서생면·온양읍에 접한다. 대한민국 남동단의 관문으로 서울특별시에서 남동쪽으로 약 450km, 대한해협을 끼고 일본 시모노세키[下關]와 약 250km 떨어져 있다. 1군 15구로 이루어져 있으며, 면적은 765.94㎢이다.
