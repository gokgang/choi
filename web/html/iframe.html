HTML Iframe

HTML iframe 은 웹 문서 안에 웹 페이지를 표시한다.

Iframe 구문

HTML <iframe> 요소는 Inline Frame 을 지정한다.
Inline Frame은 현재 HTML 문서 안에 다른 문서를 포함한다.

<iframe src="url" title="description"></iframe>

<iframe> 의 title 속성은 Screen Reader 에서 <iframe> 의 내용을 읽는 데 사용한다.

높이와 너비 설정

높이와 너비는 height 및 width 를 사용하여 픽셀 단위를 기본으로 iframe 의 크기를 지정한다.

<iframe src="iframe_example.htm" height="200" width="300" title="Iframe 예제"></iframe>
<iframe src="iframe_example.htm" style="height:200px;width:300px" title="Iframe 예제"></iframe>

테두리 없애기

iframe 은 기본적으로 테두리가 있다. 테두리를 제거하려면 style 속성에서 border 속성값을 지정하면 된다.

<iframe src="iframe_example.htm" style="border:none;height:200px;width:300px;" title="Iframe 예제"></iframe>

링크 대상

iframe 의 name 속성을 지정하면 name 을 링크의 대상 프레임으로 사용할 수 있다. 

    <p><a href="/" target="mainframe">Home</a></p>
    <iframe src="iframe_example.htm" style="border:none;height:300px;width:100%;" title="Iframe 예제" name="mainframe">
    </iframe>

Home 원래대로 돌아가려면 새로고침 하세요
