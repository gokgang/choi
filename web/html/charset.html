HTML Encoding (Character Sets)

HTML 페이지를 올바르게 표시하려면 웹 브라우저가 사용할 문자 모음(Character Sets)을 알아야 한다.

ASCII에서 UTF-8로

ASCII는 최초의 문자 인코딩 표준이었습니다. ASCII는 인터넷에서 사용할 수 있는 128 개의 다른 문자를 정의했다.
숫자 (0-9), 영문자(A-Z a-z) , ! $ + - () @ < > .

ISO-8859-1은 HTML 4의 기본 문자 모음이었다.이 문자 모음은 256개의 서로 다른 문자 코드를 지원했으며, HTML 4는 UTF-8도 지원했다.

ANSI (Windows-1252)는 원래 Windows 문자 집합이었다. ANSI는 32개의 추가한 문자가 있다는 점을 제외하면 ISO-8859-1과 같다.

HTML5 사양은 웹 개발자가 세계의 거의 모든 문자와 기호를 포함하는 UTF-8 문자 모음을 사용하도록 권장한다!

HTML charset 속성

웹 브라우저가 HTML 문서를 올바르게 표시하려면 문서에 사용된 문자 모음을 알고 있어야 한다.

<meta charset="UTF-8">
