h1 ~ h6 요소

h1 ~ h6 요소는 웹 문서에 제목을 표시할 때 사용한다. 

  <h1>Heading 1</h1>
  <h2>Heading 2</h2>
  <h3>Heading 3</h3>
  <h4>Heading 4</h4>
  <h5>Heading 5</h5>
  <h6>Heading 6</h6>

<h1> ~ <h6> 요소는 HTML 문서에서 제목을 작성할 때 사용하는 요소이다.
<h1> 은 가장 중요한 기본 제목을 작성할 때 사용하고, <h6> 까지 점차적으로 중요성에 따른 제목 작성에 사용한다.
제목의 앞 뒤에는 공백을 추가하여 한줄을 모두 차지한다.
제목 요소는 굵게 표시되므로 문서의 내용에서 굵게 표시하기 위하여 이 요소를 사용해서는 않된다.

검색 엔진은 제목을 사용하여 웹 페이지의 구조와 내용을 색인으로 만든다.
사용자들은 웹 문서의 제목을 보고 문서의 내용을 짐작하기 때문에 문서의 구조와 내용에 적합한 제목을 사용하는 것은 중요하다.

제목 요소는 기본 크기가 정해져 있지만 style 속성의 CSS font-size 속성을 사용하여 글자 크기를 다시 설정할 수 있다. 

<h1  style="font-size: 64px;">Heading 1</h1>
