HTML 문단

문단은 항상 새로운 줄에서 시작하며 일반적인 글의 묶음이다.

차례 : 문단 p 수평선 hr 줄 바꿈 br 글 모양 유지 pre

문단(Paragraph)

HTML 의 <p> 요소는 하나의 문단을 정의한다.
문단은 항상 새로운 줄에서 시작되고 웹 브라우저는 문단의 앞 뒤에 자동으로 공백(margin)을 추가한다.

HTML 화면 표시

HTML 문서가 화면에 어떻게 표시되는지 확신할 수 없다.
크거나 작은 화면 및 크기가 조정된 창은 다른 결과를 생성한다.
HTML을 사용하면 HTML 문서에 공백이나 줄을 추가하여 표시를 변경할 수 없다.
브라우저는 웹 문서가 표시될 때 추가한 공백과 줄 바꿈을 제거한다.

다음의 문장은 보이는 그대로 화면에 표시되지 않는다. 

  <p>
    이 문단은
    여러 줄로 되어 있다.
    웹 문서를 작성할 때,
    여러 개의 공백과 줄로 만들어도
    웹 브라우저는
    그것을 무시한다.
  </p>

이 문단은 여러 줄로 되어 있다. 웹 문서를 작성할 때, 여러 개의 공백과 줄로 만들어도 웹 브라우저는 그것을 무시한다.

수평선( hr )

<hr> 요소는 HTML 문서에서 주제를 변경할 때 자주 사용하며 수평선으로 표시된다. 이 <hr>요소는 HTML 문서에서 내용을 분리하거나 새로운 정의를 하는 데 사용된다.

<hr> 요소는 끝 매듭이 없는 빈 매듭이다.

줄 바꿈( br )

<br>요소는 줄 바꿈을 지정한다.
새로운 문단을 시작하지 않고 새 줄에서 시작하려고 할 때 사용한다. 

  <p>This is<br>a paragraph<br>with line breaks.</p>
<!-- 결과 : -->
This is
a paragraph
with line breaks.

<br> 요소는 끝 매듭이 없는 빈 매듭이다.

<pre> 요소

<pre>요소 안의 글은 고정 너비 글꼴 (일반적으로 Courier)로 표시되며 공백과 줄 바꿈을 모두 그대로 유지한다. 

  <pre>
  나 보기가 역겨워
  가실 때에는
  말없이 고이 보내드리오리다
  영변에 약산
  진달래꽃,
  아름 따다 가실 길에 뿌리오리다

  가시는 걸음걸음
  놓인 그 꽃을
  사뿐히 지르밟고 가시옵소서

  나 보기가 역겨워
  가실 때에는
  죽어도 아니 눈물 흘리오리다
  </pre>
