HTML id 속성

HTML 요소의 고유 식별자는 id 속성으로 지정한다. HTML 문서에서 동일한 식별자( id )를 가진 요소를 두 개 이상 가질 수 없다.

id 속성 지정

id 속성 값은 HTML 문서 내에서 고유해야 하며, CSS에서 특정 양식 선언에 사용된다. 또한 JavaScript에서 특정 식별자( id )를 가진 요소에 접근하고 조작하는 데 사용된다.
CSS id 구문은 식별자 이름 앞에 hash (#)를 붙이고, 이어서 중괄호 {} 안에 양식 속성을 선언한다. 

  <!DOCTYPE html>
  <html>
  <head>
    <style>
        #header {
            background-color: lightblue;
            color: black;
            padding: 40px;
            text-align: center;
        }
    </style>
  </head>
  <body>
  
    <h1 id="header">My Header</h1>
  
  </body>
  </html>

My Header

식별자( id )는 대·소문자를 구분하며, 하나 이상의 문자를 포함해야 하고, 숫자로 시작할 수 없으며, 공백을 포함할 수 없다. 

  <!DOCTYPE html>
  <html>
  <head>
  <style>
    /* id="header" 요소의 양식 */
    #header {
      background-color: lightblue;
      color: black;
      padding: 40px;
      text-align: center;
    }

    /* class="city" 요소의 양식 */
    .city {
      background-color: darkcyan;
      color: white;
      padding: 10px;
    }
    </style>
    </head>
    <body>

      <h1 id="header">한국의 도시</h1>
      <h2 class="city">서울</h2>
      <p>서울은 대한민국의 수도이다.</p>

      <h2 class="city">부산</h2>
      <p>부산은 대한민국의 제일의 항구 도시이다.</p>

      <h2 class="city">인천</h2>
      <p>인천은 대한민국의 관문인 인천국제공항이 있다.</p>

  </body>
  </html>

한국의 도시

서울

서울은 대한민국의 수도이다.

부산

부산은 대한민국의 제일의 항구 도시이다.

인천

인천은 대한민국의 관문인 인천국제공항이 있다.

HTML 책갈피

HTML 책갈피( bookmark )는 사용자가 웹 페이지의 특정 부분으로 이동할 수 있도록 하는 데 사용된다.
문서가 매우 긴 경우 책갈피가 유용할 수 있다.
책갈피를 사용하려면 먼저 id 를 지정하여 책갈피를 만든 다음 링크를 추가한다.
그런 다음 링크를 클릭하면 페이지가 채갈피가 있는 위치로 이동한다.

id 속성 지정

  <h2 id="attribute">id 속성 지정</h2>

link 지정

  <a href="#attribute">id 속성 지정</a>

JavaScript에서 id 속성 사용

id 속성은 특정 요소에 대한 작업을 수행하는 JavaScript에 사용할 수 있다.
JavaScript는 getElementById() 메서드를 사용하여 특정 id 를 가진 요소에 접근할 수 있다 . 

  <script>
    function changedisplay () {
      document.getElementById("disp").innerHTML = "자바 스크립트에서 HTML 요소에 접근";
    }
  </script>
  <button onclick="changedisplay ()">제목 바꾸기</button>
