이모티콘

이모티콘은 UTF-8 글자 모임의 글자이다. 😄 😍 💗

이모티콘은 그림이나 아이콘처럼 보이지만 Unicode UTF-8 글자 모임의 글자이다.

UTF-8은 세계의 거의 모든 글자와 기호를 정해두고 있다.

charset 속성

HTML 페이지를 올바르게 표시하려면 웹 브라우저가 페이지에 사용된 글자 모임을 알고 있어야 한다.

글자 모임은 <meta> 요소로 지정한다.

  <meta charset="UTF-8">

글자 모임을 지정하지 않으면 기본값인 UTF-8 을 적용한다

UTF-8 글자

많은 UTF-8 글자는 키보드로 입력할 수 없지만 항상 숫자 (글자 번호, entity numbers)를 사용하여 나타낼 수 있다.

  • A : 65
  • B : 66
  • C : 67
  <!DOCTYPE html>
  <html>
  <head>
    <meta charset="UTF-8">
  </head>
  <body>

  <p>&#65; &#66; &#67;</p>

  </body>
  </html>

A B C

예제 설명

<meta charset="UTF-8"> 요소는 유니코드 글자 모임을 지정한다.

<p>&#65; &#66; &#67;</p>는 영문 대문자 A B C 를 나타낸다.
글자 번호 앞에 Ampersand(&) 와 Hash (#) 를, 글자 번호 뒤에는 Semicolon( ; ) 을 붙여서 Unicode 글자임을 Web browser 에게 알려준다.
