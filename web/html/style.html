HTML style(양식) 속성

HTML style 속성은 색상, 글꼴, 크기 등과 같은 양식을 설정한다.
HTML 요소에 style 속성을 사용하여 양식을 설정하면 따로 작성된 CSS 양식이 있더라도 우선 적용된다. 그러나 문서의 요소에 직접 양식을 지정하면 번거로울 뿐만 아니라 문서가 매우 혼잡스러워진다. 그러므로 특정한 요소의 양식만 따로 설정할 필요가 있을 때만 사용하는 것이 좋다.

차례 : 바탕색 | 글자색 | 글꼴 | 글자 크기 | 가로 맞춤 
  <p style="color:red;">I am red</p>
  <p style="color:blue;">I am blue</p>
  
  <p style="font-size:64px;">I am big</p>

HTML 요소의 style 속성

HTML 요소의 양식 설정은 style 속성으로 설정한다 .

구문(Syntax)

<요소이름 style="속성: 값;">

속성과 값은 CSS의 속성과 값이다.

바탕색

CSS background-color 속성은 HTML 요소의 바탕색을 설정한다.

<!-- 문서의 바탕색 설정 : powderblue --> 
  <body style="background-color: powderblue;">
    ...
  </body>
<!-- 다른 두 요소의 바탕색 설정 : powderblue --> 
  <h1 style="background-color: powderblue;">주요 제목</h1>
  <p style="background-color: mediumpurple;">이 글은 예제를 위한 문단이다.</p>

본문의 주 제목

이 글은 예제를 위한 문단이다.

글자색

CSS color 속성은 HTML 요소의 글자색을 설정한다. 

  <h1 style="color: blue;">주요 제목</h1>
  <p style="color: brown;">이 글은 예제를 위한 문단이다.</p>

주요 제목

이 글은 예제를 위한 문단이다.

글꼴

CSS font-family 속성은 HTML 요소에 사용할 글꼴을 설정한다. 

  <h1 style="font-family: verdana;">이 제목의 글꼴은 verdana 이다</h1>
  <p style="font-family: courier;">이 글의 글꼴은 courier 이다.</p>

이 제목의 글꼴은 verdana 이다

이 글의 글꼴은 courier 이다.

글자 크기

CSS font-size 속성은 HTML 요소에 사용할 글자 크기를 설정한다. 

  <h1 style="font-size: 300%;">이 제목의 글자 크기는 300%이다</h1>
  <p style="font-size: 160%;">이 글의 글자 크기는 160% 이다.</p>

이 제목의 글자 크기는 300% 이다

이 글의 글자 크기는 160% 이다.

가로 맞춤

CSS text-align 속성은 HTML 요소의 글자 가로 맞춤을 설정한다. 

  <h1 style="text-align: center;">이 제목은 가로 가운데 맞춤으로 정렬한다</h1>
  <p style="text-align: center;">이 글은 가로 가운데 맞춤으로 정렬한다.</p>

이 제목은 가로 가운데 맞춤으로 정렬한다

이 글은 가로 가운데 맞춤으로 정렬한다.

정리

    HTML 요소의 양식은 style 속성을 사용하여 설정한다.

  • 바탕색은 background-color 사용
  • 글자색은 color 속성 사용
  • 글꼴은 font-family 속성 사용
  • 글자 크기는 font-size 속성 사용
  • 글 가로 맞춤 text-align 속성 사용
