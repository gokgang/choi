표 양식

표 테두리 안 띄우기 표제 왼쪽 맞춤 테두리 틈 지정
가로 행 칸 합치기 세로 열 칸 합치기 표제 넣기 독특한 표

표 만들기는 <table> 요소를 사용하여 설정한다.
표의 가로 행은 <tr> 매듭으로 지정하고, 표의 표제 줄은 <th> 매듭으로 지정하며, 표의 칸은 <td> 매듭으로 지정한다.
<th> 요소는 기본적으로 굵은 글자에 가운데 맞춤이 되며, <td> 요소는 기본적으로 왼쪽 맞춤이 된다. 

<!-- 표 만들기 지정 -->
  <table style="width: 100%;">
    <tr>  <!-- 가로 행 지정 -->
      <th>성</th><th>이름</th><th>나이</th>  <!-- 표제 지정 -->
    </tr>
    <tr>  <!-- 가로 행 지정 -->
      <td>반</td><td>기문</td><td>70</td>   <!-- 각 칸에 내용 입력 -->
    </tr>
    <tr>  <!-- 가로 행 지정 -->
      <td>문</td><td>재인</td><td>69</td>   <!-- 각 칸에 내용 입력 -->
    </tr>
  </table>

<td> 요소는 표의 칸으로 글과 그림, 목록 등 다른 모든 자료를 담을 수 있는 그릇과 같다.

표 테두리 border

표의 테두리는 CSS border 속성을 사용하여 지정한다. 

  <style>
    table, th, td {
      border: 1px solid skyblue;
    }
  </style>

테두리는 표의 바깥 테두리와 각 칸의 테두리를 모두 설정해야 한다. 각각의 테두리 사이에는 틈이 있다. 틈을 없애려면 별도의 설정이 필요하다. 

  <style>
    table, th, td {
      border: 1px solid skyblue;
      border-collapse: collapse;
    }
  </style>

안 띄우기

테두리와 내용 사이에 띄우기를 padding 속성으로 지정할 수 있다.

    table, th, td {
      border: 1px solid skyblue;
      border-collapse: collapse;
      padding: 16px;

가운데 맞춤으로된 표제를 text-align 속성을 지정하여 왼쪽으로 맞출 수 있다. 

  <style>
    table, th, td {
      border: 1px solid skyblue;
      border-collapse: collapse;
      padding: 16px;
      text-align: left;

독특한 표 양식

자기 만의 독특한 표 양식을 지정하려면 id 속성을 설정한다. 

      <style>
        #special {
          width: 100%;
          background-color: #f1f1c1;
          border: 1px solid skyblue;
          border-collapse: collapse;
        }
        #special th, #special td {
          border: 1px solid skyblue;
          padding: 6px;
          text-align: left;
        }
        #special tr:nth-child(even) {
          background-color: #def;
        }
        /* #special tr:nth-child(odd) {
          background-color: #fff;
        } */
        #special th {
          color: yellow;
          background-color: navy;
        }
      </style>
