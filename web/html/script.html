JavaScript

JavaScript는 HTML 문서를 보다 동적이고 대화식으로 만들어 준다.

<script> 요소

HTML <script> 요소는 사용자의 브라우저에서 실행되는 Script(JavaScript)를 정의한다.
<script> 매듭 안에 JavaScript 문장을 작성하거나 src 속성을 사용하여 외부 Script 파일을 포함시킬 수 있다.
JavaScript 는 일반적으로 그림의 조작, 양식의 유효성 검사, 요소 내용의 설정 등을 수행한다.
JavaScript는 document.getElementById( “id” ) 메서드를 사용하여 HTML 요소를 선택한다.

JavaScript 수행 작업 예제

요소 내용 변경

  <script>
    document.getElementById("demo").innerHTML = "Hello JavaScript!";
  </script>
  <p id="demo"></p>

양식 변경

  <script>
    document.getElementById("demo_1").style.fontSize = "25px";
    document.getElementById("demo_1").style.color = "red";
    document.getElementById("demo_1").style.backgroundColor = "yellow";
  </script>
  <p id="demo_1"></p>

HTML 요소의 속성 변경

  <script>
    document.getElementById("image").src = "image/img_flower.gif";
  </script>
  <img id="image" src="image/img_firstlove.jpg" alt="꽃말 첫사랑">

<noscript> 요소

<noscript> 요소는 브라우저에서 스크립트를 비활성화했거나 스크립트를 지원하지 않는 브라우저가 있는 사용자에게 표시할 대체 내용을 정의한다. 

  <script>
    document.getElementById("demo").innerHTML = "Hello JavaScript!";
  </script>
  <noscript>미안하지만, 이 브라우저는 자바 스크립트를 지원하지 않습니다!</noscript>
