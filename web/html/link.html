HTML Links

구문 열림창(target) 그림 연결 메일 연결 풍선도움말 색상 연결 단추 책갈피

HTML Links 는 Hyperlink 이다.

링크 위로 마우스를 이동하면 마우스 화살표가 작은 손 모양으로 바뀐다.
링크를 클릭하면 다른 문서로 이동할 수 있으며, 글자 뿐만 아니라 그림, 기타 HTML 요소에도 링크를 지정할 수 있다.

구문(Sntax)

<a> 요소( Anchor )는 href 속성을 사용하여 다른 문서나 같은 문서 안의 어느 위치, 파일, 이메일 주소와 그 외 다른 URL로 연결할 수 있는 하이퍼링크를 지정한다. <a> 매듭 안의 내용은 링크된 목적지의 설명을 나타낸다.

<a href="url" target="_self | _blank | _parent | _top"> link text </a>

target 속성

기본적으로 링크된 문서는 현재 브라우저 창에 표시된다.

target 속성은 다음 중 하나를 지정할 수 있다.

  • _self - 현재 창. 기본값
  • _blank - 브라우저 설정에 따라 새탭 또는 새창
  • _parent - 상위 프레임. 상위 창이 없으면 _self 와 동일
  • _top - 최상위 창. 자신이 최상위이면 _self 와 동일

참고: target 을 사용할 때, rel="noreferrer"를 추가해 window.opener API 의 악의적인 사용을 방지할 수 있다.
최근의 브라우저(Firefox 79+ 등)에서는 target="_blank"를 지정하면 rel="noopener"를 적용한 것과 같은 동작을 한다.

절대 URL과 상대 URL

href 속성의 url 은 하이퍼링크가 가리키는 URL. 링크는 HTTP 기반 URL일 필요는 없고, 브라우저가 지원하는 모든 URL 을 사용할 수 있다. 절대 URL(전체 웹 주소)과 상대 URL(동일한 웹 사이트 내의 페이지에 대한 링크)을 모두 사용할 수 있다.

그림에 연결 설정

그림을 연결에 사용하려면 <a> 요소 안에 <img> 요소를 넣으면 된다. 

  <a href="default.html">
    <img src="img_flwr.gif>" width="80">
  </a>

전자 메일 연결

href 속성에 mailto: 속성값을 설정하면 사용자의 전자메일 프로그램을 작동하게 할 수 있다.

<a href="mailto:someone@example.com">전자 메일 보내기</a>

연결 단추(Link button)

JavaScript를 사용하여 버튼 클릭과 같은 특정 사건에서 발생하는 작업을 지정할 수 있다. 

  <button onclick="document.location='default.html'">Go Home<button>  

  <button onclick="window.location='default.html'">Go Home<button>  

  <button onclick="location='default.html'">Go Home<button>

title 속성

<a> 요소에 title 속성을 지정하면 마우스가 올려졌을 때 풍선 도움말이 보여진다. 

  <a href="default.html" title="홈으로" style="border: 1px solid grey;">Go Home</a>  Go Home

Links 색상

기본적으로 링크는 모든 브라우저에서 다음과 같이 나타난다.

  • 방문하지 않은 링크( unvisited link )는 밑줄과 파란색( blue )이다.
  • 방문한 링크( visited link )는 밑줄과 자주색( purple )으로 표시된다.
  • 활성 링크( active link )는 밑줄과 빨간색( red )으로 표시된다.

링크는 CSS로 다른 양식을 지정할 수 있다. 

  <style>
    a:link {  /* 방문하지 않은 링크 */
      color: blue;
      background-color: transparent;
      text-decoration: none;
    }
    a:visited { /* 방문한 링크 */
      color: magenta;
      background-color: transparent;
      text-decoration: none;
    }
    a:hover { /* 링크 위에 마우스가 놓일 때 */
      color: red;
      background-color: transparent;
      text-decoration: underline;
    }
    a:active {  /* 활성 링크 */
      color: yellow;
      background-color: transparent;
      text-decoration: underline;
    }
  </style>

연결 단추

CSS를 사용하여 링크 양식을 단추로 지정할 수도 있다. 

  <style>
    a:link, a:visited {
      background-color: grey;
      color: white;
      padding: 6px 16px;
      text-align: center;
      text-decoration: none;
      border: 1px solid grey;
      border-radius: 2px;
      display: inline-block;
    }
    a:hover, a:active {
      background-color: lightgrey;
      color: blue;
    }
  </style>

책갈피 만들기

HTML 링크를 사용하여 사용자가 웹 문서의 특정 부분으로 이동할 수 있도록 할 수 있다.
웹 문서가 매우 긴 경우 먼저 id를 설정하여 책갈피를 만든 다음 링크를 추가해 두면, 링크를 클릭했을 때 문서의 채갈피가 있는 위치로 이동한다

책갈피 만들기

<h2 id="heading">문서의 처음으로</h2>

책갈피에 대한 링크
