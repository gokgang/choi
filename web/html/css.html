HTML 양식

HTML의 양식을 지정하는 CSS는 Cascading Style Sheets를 의미한다.
CSS는 많은 작업을 줄여주고 여러 개의 웹 문서를 같은 양식을 갖도록 할 수 있다.

CSS 양식

CSS를 사용하면 색상, 글꼴, 글자 크기, 요소 사이의 간격, 요소 배치 및 배치 방법, 사용할 배경 이미지 또는 배경 색상, 다양한 장치 및 화면 크기에 대해 훨씬 더 다양하게 화면을 제어할 수 있다.

계단식(Cascading)이라는 말은 상위 요소에 적용된 양식이 상위 요소 내의 모든 하위 요소에도 적용됨을 뜻한다. 따라서 본문 글자의 색상을 "파란색"으로 설정하면 따로 지정하지 않는 한 본문 내의 모든 제목, 단락 및 기타 글자도 같은 색상을 갖게 된다 !

CSS 사용

CSS는 3가지 방법으로 HTML 문서에 추가할 수 있다.

  • 요소에 포함(inline style) - HTML 요소에 style 속성에 지정
  • 문서의 내부(internal style) 글머리에 <style> 요소 지정
  • 외부(emternal style)에 작성된 양식 파일을 <link> 매듭으로 연결

Inline CSS

HTML 요소에 style 속성으로 지정한 양식은 그 요소만의 고유 양식을 적용하려고 할 때 사용한다.

다음은 <h1> 요소의 글자색을 파란색(blue)으로 하고 <p> 요소의 글자색을 빨간색(red)으로 설정한다. 

    <h1 style="color: blue;">파란색 제목</h1>
    <p style="color: red;">이 글은 빨간색으로 보여진다.</p>

파란색 제목

이 글은 빨간색으로 보여진다.

Internal CSS

문서의 <head>에 <style>을 지정하면 그 문서 만의 고유 양식이 적용된다.

다음은 문서의 바탕색을 하늘색(skyblue)으로 설정하고, <h1> 은 글자색을 파란색(skyblue)으로 설정하며, 모든 요소의 글자색은 빨간색(red)으로 설정한다. 

  <!DOCTYPE html>
  <html>
  <head>
  <style>
      body {background-color: skyblue;}
      h1 {color: blue;}
      p {color: red;}
  </style>
  </head>
  <body>
      <h1>문서 제목</h1>
      <p>이 글은 예문이다.</p>
  </body>
  </html>

External CSS

외부의 양식 파일을 사용하려면 <head> 부에 <link> 요소를 사용하여 추가한다. 

  <!DOCTYPE html>
  <html>
  <head>
      <link rel="stylesheet" href="styles.css">
  </head>
  <body>
      <h1>문서 제목</h1>
      <p>이 글은 예문이다.</p>
  </body>
  </html>

별도의 양식은 text 편집기로 작성하며, HTML 코드를 포함해서는 않된다. 양식을 정의한 파일의 확장자는 *.css 이다.

"style.css" 파일 예제

  body {
    background-color: skyblue;
  }
  h1 {
    color: blue;
  }
  p {
    color: red;
  }

참고 : 외부 양식 파일을 사용하면 하나의 파일을 변경하여 전체 웹 문서의 양식을 변경할 수 있다!

색상, 글꼴 및 크기

color 속성은 글자색 설정, font-family 속성은 글꼴 설정, font-size 속성은 글자 크기를 설정한다. 

  <!DOCTYPE html>
  <html>
  <head>
  <style>
      h1 {
        color: blue;
        font-family: serif;
        font-size: 300%;
      }
      p {
        color: red;
        font-family: sans-serif;
        font-size: 160%;
      }
  </style>
  </head>
  <body>
      <h1>문서 제목</h1>
      <p>이 글은 예문이다.</p>
  </body>
  </html>

테두리

HTML 요소에 테두리는 border 속성을 사용하여 설정한다. 

  p {
    border: 2px solid skyblue;
  }

안 띄우기 (padding)

padding 속성은 요소의 내용과 테두리 사이의 띄우기를 설정한다. 

  p {
    border: 2px solid skyblue;
    padding: 36px;
  }

바깥 띄우기 (margin)

margin 속성은 요소의 테두리 바깥의 띄우기를 설정한다. 

  p {
    border: 2px solid skyblue;
    margin: 50px;
  }

외부 양식 파일의 연결

전체 URL을 사용하여 양식 파일 연결

<link rel="stylesheet" href="https://www.example.com/css/styles.css">

현재 사이트의 css 폴더에 있는 양식 파일 연결

<link rel="stylesheet" href="/css/styles.css">

문서와 같은 폴더에 있는 양식 파일 연결

<link rel="stylesheet" href="styles.css">
