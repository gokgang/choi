서식(Text Formatting)

HTML에는 특별한 의미를 가진 글(Text)을 설정하기 위한 여러 가지 요소가 포함되어 있다.

이 글은 굵은 글자 이다.

이 글은 기울임 글자 이다.

이 글은 아래 첨자 와 위 첨자 이다.

서식 요소

  • b
    		• : 굵게특별한 중요성없이 굵은 글자
  • strong
    		• : 중요매우 중요한 글로 굵게 표시
  • i
    		• : 기울임대체 음성 또는 분위기로 글의 일부를 기울임 꼴로 표시
  • em
    		• : 강조강조된 글을 기울임 꼴로 표시
  • mark
    		• : 표식표식을 하거나 강조 표시해야 하는 글자를 지정
  • small
    		• : 작게 더 작은 글자로 지정
  • del
    		• : 지움 삭제된 글자에 한 줄을 표시
  • ins
    		• : 삽입 삽입된 글자에 밑줄 표시
  • sub
    		• : 아래 첨자 일반줄 아래에 절반 문자로 작은 글꼴로 나타냄
  • sup
    		• : 위 첨자 일반 줄 위에 절반 문자로 작은 글꼴로 나타냄
