Block 요소와 Inline 요소

Block level 요소 Inline 요소 <div> 요소 <span> 요소

모든 HTML 요소는 요소 유형에 따라 Block 과 Inline 의 두 가지 기본 표시값이 있다

Block level 요소

Block level 요소는 항상 새 줄에서 시작한다.
Block level 요소는 왼쪽과 오른쪽으로 가능한 확장하여 항상 사용 가능한 전체 너비를 차지한다.
Block level 요소에는 위쪽과 아래쪽 여백이 있지만 Inline 요소에는 여백이 없다.

<div> 요소는 Block level 요소이다. 

  <div style="border: 1px solid skyblue">Hello HypherText!</div>
Hello HypherText!

HTML의 Block level 요소

<address> <article> <aside> <blockquote> <canvas> <dd>
<div> <dl> <dt> <fieldset> <figcaption> <figure>
<footer> <form> <h1>~<h6> <header> <hr> <li>
<main> <nav> <noscript> <ol> <p> <pre>
<section> <table> <tfoot> <ul> <video>  

Inline 요소

Inline 요소는 필요한 만큼만 너비를 차지하고 새 줄에서 시작하지 않는다. 

  <p>다음의 <span style="border: 1px solid skyblue;">Hello HypherText</span>는 문단 안의 요소이다.</p>

다음의 Hello HypherText는 문단 안의 요소이다.

Inline 요소

<a> <abbr> <acronym> <b> <bdo> <big>
<br> <button> <cite> <code> <dfn> <em>
<i> <img> <input> <kbd> <label> <map>
<object> <output> <q> <samp> <script> <select>
<small> <span> <strong> <sub> <sup> <textarea>
<time> <tt> <var>      

참고 : Inline 요소는 Block Level 요소를 포함할 수 없다!

<div> 요소

<div> 요소는 다른 HTML 요소를 담을 수 있는 그릇이다.
<div>요소는 속성이 없어도 되지만, 포함된 요소들을 묶음으로 만들어 style, class, id 등을 지정하여 같은 양식으로 만들 수 있다. 

  <div  style="background-color:blue; color:white; padding:20px;">
    <h2 style="color: yellow;">서울</h2>
    <p>
      대한민국의 수도인 서울을 지방자치단체인 특별시로 부르는 명칭이다. 한반도 중앙에 있으며, 한강을 사이에 
      두고 남북으로 펼쳐져 있다. 북쪽 끝은 도봉구 도봉동, 동쪽 끝은 강동구 상일동, 남쪽 끝은 서초구 원지동, 서쪽 
      끝은 강서구 오곡동이다. 시청은 중구 을지로1가(태평로1가 31)에 있다.
    </p>
  </div>

서울

대한민국의 수도인 서울을 지방자치단체인 특별시로 부르는 명칭이다. 한반도 중앙에 있으며, 한강을 사이에 두고 남북으로 펼쳐져 있다. 북쪽 끝은 도봉구 도봉동, 동쪽 끝은 강동구 상일동, 남쪽 끝은 서초구 원지동, 서쪽 끝은 강서구 오곡동이다. 시청은 중구 태평로1가에 있다.

<span> 요소

<span>요소는 글자, 또는 문서 일부를 다르게 표시하는 데 사용되는 Inline 그릇이다.
다른 요소에 포함된 <span> 요소는 속성이 없지만, style, class, id 등을 지정하여 포함한 요소와는 다른 모양으로 만들 수 있다. 

  <p>
    어머니는 <span style="color:blue;font-weight:bold;">파란</span> 눈이고, 아버지는 
    <span style="color:darkgreen;font-weight:bold;">옅은 녹색</span> 눈을 가졌습니다.
  </p>

어머니는 파란 눈이고, 아버지는 짙은 녹색 눈을 가졌습니다.
