HTML 기호 글자

HTML에서 예약된 글자는 기호 글자로 대체되어야 한다.

기호 글자

일부 문자는 HTML로 예약되어 있다. 글 안에 보다 작음 ( < ) 또는 보다 큼 ( > ) 기호를 사용하면 브라우저에서 매듭( Tag )과 혼동할 수 있다. 기호 글자는 HTML에서 예약된 문자를 표시하는 데 사용된다. 

  &기호이름;
  또는
  &#기호번호;

보다 작음 기호 ( < )를 표시하려면 ＆lt; 또는 ＆#60; 과 같이 작성한다.

기호 이름 사용의 장점은 기호 이름을 기억하기 쉽다는 것이다. 그러나 브라우저가 모든 기호 글자 이름을 지원하지 않을 수 있으며 기호 글자 번호에 대한 지원은 좋다.

띄움 기호(Non-breaking Space)

개발자가 편집기에서 여러 개의 빈 곳을 두어도 웹 브라우저는 이를 무시하고 하나의 빈 곳만 나타낸다. 개발자가 여러 개의 빈 곳을 두기 원할 때 빈 곳 기호 글자(＆nbsp;)를 사용할 수 있다.

참고 : 줄 바꿈하지 않는 hyphen( ＆#8209; )은 새 줄로 구분되지 않는 hyphen 글자 (‑)를 정의하는 데 사용한다.

분음 부호 결합

분음 부호는 글자에 더해진 부호이다.

억음( ̀ ) , 예음( ́ ) 과 같은 분음 부호를 Accent 라고 한다.

분음 부호는 문자 위와 아래, 문자 내부 및 두 문자 사이에 나타날 수 있다.
분음 부호는 영숫자, 글자와 함께 사용하여 문서에 사용된 문자 집합 ( encoding )에 없는 문자를 생성할 수 있다.

부호글자구성결과
̀a a＆#768; à
́a a＆#769; á
̂a a＆#770; â
̃a a＆#771; ã
̀O a＆#768; à
́O a＆#769; á
̂O a＆#770; â
̃O a＆#771; ã

참고 : 특수 기호 키보드 간편 입력

MS 윈도우에서 편집기를 사용하는 경우에 간편 입력을 사용할 수 있다.
먼저 한글 자음 하나를 입력하고 한자 키 또는 우측 Ctrl 키를 누르면 특수 기호 간편 입력 선택창이 나타난다.
ㄱ ~ ㅎ 까지 각기 다른 기호 선택창이 나타나므로 편하게 입력할 수 있다.
