목록을 사용하면 개발자가 목록을 관련 항목의 묶음으로 만들 수 있다.

순서없는 목록

순서없는 목록은 <ul> 매듭으로 시작하며, 각 목록 항목은 <li> 매듭으로 묶는다.
목록 항목은 기본적으로 작고 검은 원판의 글 머리 기호가 보여진다. 

  <ul>
    <li>커피</li>
    <li>홍차</li>
    <li>우유</li>
  </ul>
  • 커피
  • 홍차
  • 우유

설명 목록

설명 목록은 <dl> 매듭은 설명 목록을 지정하며, <dt> 매듭은 용어 이름 지정 <dd> 매듭은 용어 설명을 지정한다. 

  <dl>
    <dt>커피</dt>
    <dd>- 따뜻한 음료</dd>
    <dt>우유</dt>
    <dd>- 시원한 음료</dd>
  </dl>
커피
- 따뜻한 음료
우유
- 시원한 음료

목록 매듭

매듭설 명
<ul>순서없는 목록( unordered list )
<ol>순서있는 목록( ordered list )
<li>목록 항목( list item )
<dl>설명 목록 지정( description list )
<dt>용어 지정( description term )
<dd>용어 설명( describes description )

순서없는 목록 머리 기호

<ul>매듭은 순서없는 목록을 지정한다.
목록의 머리 기호는 CSS의 list-style-type 속성으로 지정한다.

    가로 목록

    목록은 CSS를 사용하여 다양한 방식으로 양식을 지정할 수 있다.

    가로 탐색 목록

      <!DOCTYPE html>
  <html>
    <head>
      <style>
        ul {
          list-style-type: none;
          margin: 0;
          padding: 0;
          overflow: hidden;
          background-color: darkblue;
        }
        li {
          float: left;
        }
        li a {
          display: block;
          color: white;
          text-align: center;
          padding: 16px;
          text-decoration: none;
        }
        li a:hover {
          background-color: skyblue;
          color: red;
        }
      </style>
    </head>
    <body>

      <ul>
        <li><a href="#list">HTML 목록</a></li>
        <li><a href="#unordered">순서없는 목록</a></li>
        <li><a href="#ordered">순서있는 목록</a></li>

        <li><a href="#type">머리 기호</a></li>
        <li><a href="#double">목록 겹치기</a></li>
      </ul>
      
    </body>
  </html>

    순서있는 목록

    순서있는 목록은 <ol> 매듭으로 시작하며, 각 목록 항목은 <li> 매듭으로 묶는다.
    목록 항목은 기본적으로 숫자로 나타낸다. 

      <ol>
    <li>커피</li>
    <li>홍차</li>
    <li>우유</li>
  </ol>
    1. 커피
    2. 홍차
    3. 우유

    목록 type 속성

    type 속성은 <ol> 목록 항목의 번호 모양을 지정한다.

    type설 명
    type="1"숫자. 기본값
    type="A"영문 대문자.
    type="a"영문 소문자.
    type="I"대문자 로마 숫자.
    type="i"소문자 로마 숫자.
    숫자 : 
      <ol type="1">
    <li>커피</li>
    <li>홍차</li>
    <li>우유</li>
  </ol>
    1. 커피
    2. 홍차
    3. 우유
    영문 대문자 : 
      <ol type="A">
    <li>커피</li>

  </ol>

    순서 기본값 지정

    기본적으로 순서는 1부터 계산한다. 지정된 숫자부터 시작하려면 start 속성으로 지정할 수 있다 .

    시작 숫자 지정 : 
      <ol start="5">
    <li>커피</li>
    <li>홍차</li>
    <li>우유</li>
  </ol>
    1. 커피
    2. 홍차
    3. 우유

    목록 겹치기

    목록 요소는 겹치게 지정할 수 있다. 

      <ul>
    <li>커피</li>
    <li>국산차
      <ul>
        <li>녹차</li>
        <li>쌍화차</li>
      </ul>
    </li>
    <li>우유</li>
  </ul>
    • 커피
    • 국산차
      • 녹차
      • 쌍화차
    • 우유

    참고 : 목록 항목 <li> 는 새로운 목록 <ul>, 그림 <img> 링크 <a> 와 같은 HTML 요소가 포함될 수 있다.
