HTML 무리 속성(class attribuete)

HTML class 속성은 HTML 요소의 무리( class )를 지정한다.
여러가지 HTML 요소가 같은 무리( class )에 속하면 양식( style )을 공유할 수 있다.

class 속성 사용

class 속성은 CSS에서 class 이름으로 양식을 지정한다. 특정 class 이름으로 요소에 접근하고 조작하기 위해 JavaScript에서 사용할 수도 있다.

다음은 class 가 city인 속성이 있는 3개의 <div> 요소는 모두 지정된 CSS 로 양식이 표시된다. 

  <!DOCTYPE html>
  <html>
  <head>
    <style>
    .city {
      background-color: darkcyan;
      color: white;
      border: 1px solid grey;
      margin: 16px;
      padding: 16px;
    }
    </style>
  </head>
  <body>

  <div class="city">
    <h2>서울</h2>
    <p>서울은 대한민국의 수도이다.</p>
  </div>

  <div class="city">
    <h2>부산</h2>
    <p>부산은 대한민국의 제일의 항구 도시이다.</p>
  </div>

  <div class="city">
    <h2>인천</h2>
    <p>인천은 대한민국의 관문인 인천국제공항이 있다.</p>
  </div>

  </body>
  </html>

서울

서울은 대한민국의 수도이다.

부산

부산은 대한민국의 제일의 항구 도시이다.

인천

인천은 대한민국의 관문인 인천국제공항이 있다.

  <!DOCTYPE html>
  <html>
  <head>
    <style>
    .note {
      color: red;
      font-size: 120%;
    }
    </style>
  </head>
  <body>

    <h2>이 글은 매우 <span class="note">중요한 글</span>이다.</h2>
    <p>Class 이름은 <span class="note">대·소 문자 구분</span>을 한다.</p>

  </body>
  </html>

이 글은 매우 중요한 글이다.

Class 이름은 대·소 문자 구분을 한다.

CSS Class 구문

양식( Style ) 무리( Class )를 만들려면 마침표 (.) 기호와 무리 이름을 입력하고 그 다음 중괄호 {} 안에 CSS 속성( Attribute )을 선언한다. 

  <style>
    .무리-이름 {
        속성1: 속성값;
        속성2: 속성값;
    }
  </style>

여러가지 무리( Multiple Classes )

HTML 요소는 둘 이상의 무리( class )에 속할 수 있다.
여러가지 무리를 지정하려면 <div class = "city main">와 같이 무리 이름을 공백으로 구분하여 나열하면, 지정된 모든 무리의 양식을 가지게 된다. 

  <!DOCTYPE html>
  <html>
  <head>
    <style>
      .city {
        background-color: darkcyan;
        color: white;
        border: 1px solid grey;
        margin: 16px;
        padding: 16px;
      }
      .center {
        text-align: center;
      }
    </style>
  </head>
  <body>

  <h2 class="city center">서울</h2>
  <h2 class="city">부산</h2>
  <h2 class="city">인천</h2>

  </body>
  </html>

서울

부산

인천

다른 요소가 같은 무리( Class )를 공유

다른 HTML 요소가 같은 무리의 이름을 가리킬 수 있다. 같은 무리의 이름을 가지게 되면 같은 양식을 공유하게 된다. 

  <h2 class="city">서울</h2>
  <p class="city">서울은 대한민국의 수도이다.</p>

서울

서울은 대한민국의 수도이다.

JavaScript에서 무리( Class ) 속성 사용

버튼을 클릭하면 무리의 이름이 "city"인 요소를 모두 숨긴다.

  <script>
    function hideCity () {
      const class_city = document.getElementsByClassName("city");
      for (let i=0; i < class_city.length; i++) {
        class_city[i].style.display = "none";
      }
    }
  </script>
